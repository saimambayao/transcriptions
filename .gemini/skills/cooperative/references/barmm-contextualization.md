# General to BARMM Contextualization Standard

**CRITICAL**: This pedagogical standard MUST be applied to ALL cooperative curriculum design.

## The Two-Layer Approach

```
LAYER 1: GENERAL/NATIONAL CONTEXT (Always First)
  Present what is taught GENERALLY by ALL cooperatives nationwide:
    - ICA 7 Cooperative Principles (universal, 1995)
    - ICA Cooperative Values (self-help, democracy, equality, etc.)
    - RA 9520 provisions (Philippine Cooperative Code)
    - CDA regulations and memorandum circulars
    - Standard cooperative governance structures
    - Common cooperative practices nationwide

                        |
                        v

LAYER 2: BARMM CONTEXTUALIZATION (Then Second)
  Present BARMM-specific application and contextualization:
    - CSEA regulations and authority
    - BTA Bill 210 requirements
    - Islamic principles in cooperative operations
    - Shari'ah-compliant alternatives (Murabaha, Wadiah, etc.)
    - Moral Governance 12 pillars
    - BDP 2023-2028 alignment
    - Halal transaction requirements
```

## Exception: BARMM-First Delivery

When general context is not applicable or contradictory, BARMM context may be delivered first:

| Scenario | Reason | Example |
|----------|--------|---------|
| **No national equivalent** | Topic is BARMM-specific with no general counterpart | CSEA-specific registration procedures |
| **Contradictory principles** | General practice conflicts with BARMM requirements | Riba (interest) vs Shari'ah-compliant financing |
| **Regional-only programs** | Programs that only exist in BARMM | BTA Bill 210 provisions not in RA 9520 |
| **Cultural specificity** | Content inherently tied to Bangsamoro culture | Moral Governance pillars, Islamic values |

## Application to Cooperative Topics

| Topic | Layer 1 (General) | Layer 2 (BARMM) |
|-------|-------------------|-----------------|
| **Cooperative Values** | ICA values: self-help, self-responsibility, democracy, equality, equity, solidarity | Islamic values alignment: Amanah, Adalah, Shura, Ta'awun |
| **Cooperative Principles** | ICA 7 Principles (RA 9520 Art. 4) | How principles manifest in Bangsamoro context |
| **Financial Services** | Conventional credit operations (RA 9520) | Shari'ah-compliant: Murabaha, Mudarabah, Musharakah |
| **Governance** | RA 9520 governance structure | Moral Governance integration, Shura principles |
| **Registration** | CDA registration process | CSEA registration, BTA Bill 210 requirements |
| **Compliance** | CDA compliance requirements | CSEA compliance, halal certification |

## Decision Tree

```
START: Design cooperative curriculum content
    |
    +-> Does a GENERAL/NATIONAL standard exist for this topic?
    |   |
    |   +-> YES: Does it conflict with BARMM/Islamic principles?
    |   |   |
    |   |   +-> NO CONFLICT: Use Two-Layer Approach
    |   |   |   Layer 1: Present general standard first
    |   |   |   Layer 2: Show BARMM contextualization
    |   |   |
    |   |   +-> CONFLICT: Use BARMM-First
    |   |       Present BARMM-compliant alternative
    |   |       Explain why general practice differs
    |   |
    |   +-> NO: Use BARMM-First
    |       Present BARMM-specific content directly
    |       Note that this is regional-specific
    |
    +-> END
```

## Implementation Checklist

When designing cooperative curriculum:

- [ ] **Identify general standards first**: What do ALL Philippine cooperatives learn about this topic?
- [ ] **Check for conflicts**: Does the general standard conflict with Islamic/BARMM principles?
- [ ] **Structure two layers**: If no conflict, present general THEN BARMM context
- [ ] **Note exceptions**: If conflict exists, explain the BARMM-compliant alternative
- [ ] **Cross-reference sources**: Cite both RA 9520/CDA AND CSEA/BTA sources
- [ ] **Validate with /academy**: Ensure curriculum follows MoroAcademy standards
