# Lean Canvas Curriculum Design

## Integration with /academy

This reference provides curriculum design guidance for Lean Canvas training in MoroAcademy.

## Learning Levels

### Foundational Level
**Target**: New entrepreneurs, cooperative members, aspiring SE founders
**Duration**: 4-8 hours (half-day to full-day workshop)
**Outcome**: Complete first Lean Canvas draft

### Intermediate Level
**Target**: Active entrepreneurs, coop managers, SE operators
**Duration**: 16-24 hours (2-3 day workshop)
**Outcome**: Validated Lean Canvas with customer evidence

### Advanced Level
**Target**: Scaling businesses, mentors, business advisors
**Duration**: 40+ hours (week-long or multi-week program)
**Outcome**: Iteration skills, pivot decisions, mentoring capability

## Standard Course Modules

### Module 1: Introduction to Lean Canvas (2 hours)
**Learning Objectives:**
- Explain the purpose and origin of Lean Canvas
- Distinguish Lean Canvas from Business Model Canvas
- Describe the 9 building blocks

**Content:**
- Ash Maurya and Running Lean methodology
- Why startups fail (42% due to "no market need")
- Build-Measure-Learn loop
- Canvas overview and flow

**Activities:**
- Case study analysis of successful canvas
- Quick canvas fill for familiar business

### Module 2: Problem Discovery (2-3 hours)
**Learning Objectives:**
- Identify customer problems worth solving
- Distinguish symptoms from root causes
- Prioritize problems by severity

**Content:**
- Problem interview techniques
- Existing alternatives analysis
- Problem validation methods

**Activities:**
- Problem interview role-play
- Map existing alternatives for target market
- Problem prioritization exercise

### Module 3: Customer Segments (2 hours)
**Learning Objectives:**
- Define specific customer segments
- Identify early adopters
- Create customer personas

**Content:**
- Segmentation criteria
- Early adopter characteristics
- Persona development

**Activities:**
- Segment mapping exercise
- Early adopter identification
- Persona creation workshop

### Module 4: Unique Value Proposition (2 hours)
**Learning Objectives:**
- Craft compelling value propositions
- Differentiate from competitors
- Test UVP with customers

**Content:**
- UVP formula and templates
- Competitive positioning
- Message testing methods

**Activities:**
- UVP writing workshop
- Competitor analysis
- Peer feedback sessions

### Module 5: Solution Design (2-3 hours)
**Learning Objectives:**
- Design minimum viable solutions
- Prioritize features by impact
- Plan MVP experiments

**Content:**
- MVP concept and types
- Feature prioritization frameworks
- Solution interview techniques

**Activities:**
- Solution sketching
- MVP scoping exercise
- Solution interview practice

### Module 6: Channels & Revenue (2 hours)
**Learning Objectives:**
- Identify effective customer channels
- Design revenue models
- Calculate unit economics

**Content:**
- Inbound vs outbound channels
- Revenue model types
- Pricing strategies

**Activities:**
- Channel mapping
- Pricing exercise
- Revenue projection

### Module 7: Metrics & Costs (2 hours)
**Learning Objectives:**
- Select key metrics that matter
- Estimate cost structure
- Calculate runway and breakeven

**Content:**
- Vanity vs actionable metrics
- Cost categories
- Financial projections

**Activities:**
- Metric selection exercise
- Cost estimation
- Breakeven analysis

### Module 8: Unfair Advantage & Iteration (2 hours)
**Learning Objectives:**
- Identify sustainable competitive advantages
- Plan canvas iterations
- Make pivot/persevere decisions

**Content:**
- Types of unfair advantages
- Iteration best practices
- Pivot decision framework

**Activities:**
- Unfair advantage brainstorm
- Canvas review and iteration
- Pivot case studies

## Assessment Methods

### Formative Assessments
- Canvas block completions (check for understanding)
- Peer review sessions
- Interview practice feedback

### Summative Assessments
- Complete Lean Canvas presentation
- Customer evidence portfolio
- Iteration documentation

### Portfolio Components
1. Initial canvas draft
2. Customer interview notes (minimum 5 interviews)
3. Iterated canvas with changes documented
4. Key learnings reflection

## BARMM-Specific Adaptations

> **Source:** Bangsamoro Development Plan (BDP) 2023-2028

### For Cooperatives (Aligned with BDP 2023-2028)
- Add cooperative principles overlay
- Include member-owner perspective
- Integrate democratic governance considerations
- Connect to CSEA compliance requirements
- **BDP Priority:** Raise compliance from 11.66% to 50%
- Focus on Producer (37.58%), Marketing (27.54%), and Agricultural (12.82%) types

### For Social Enterprises
- Use Social Lean Canvas framework
- Include impact measurement module
- Add beneficiary vs customer distinction
- Integrate Shari'ah compliance (where applicable)
- Connect to BCSE Bill requirements (pending Parliament)
- Align with "Tatak SE for BARMM Peace and Prosperity" program

### Cultural Considerations
- Use local examples and case studies from BDP priority sectors
- Incorporate Bangsamoro success stories
- Respect Islamic business principles
- Support Bahasa Sug/Meranaw/Maguindanaon translations
- Reference moral governance principles (12 pillars from BDP)

### BDP-Aligned Problem Discovery Topics

**For Module 2 (Problem Discovery):**
Use these BDP-identified challenges:
1. Low MSME formalization (only 0.92% of national BNRs)
2. Limited access to credit/financing
3. High production and utility costs
4. Weak market linkages and connectivity
5. Low technology adoption
6. Post-harvest losses
7. Skills mismatch in workforce

**For Module 4 (Unique Value Proposition):**
Leverage BARMM competitive advantages:
1. #1 seaweed producer nationally
2. Strategic BIMP-EAGA location
3. Halal industry potential
4. Islamic banking/finance
5. Rich natural resources
6. Young, trainable workforce (42% aged 15-39)

### Investment Climate Context

**For Modules 6-7 (Channels, Revenue, Costs):**
Reference BDP investment data:
- PHP 25.3B total investments (2010-2021)
- 20,103 jobs created from registered investments
- PHP 9.02B MSME investments (2017-2021)
- 62,166 jobs from MSMEs
- Bangsamoro Board of Investments (BBOI) incentives available

## Facilitation Guide

### Pre-Workshop Preparation
1. Prepare local case studies
2. Print canvas templates (A3 size recommended)
3. Gather sticky notes and markers
4. Set up customer interview pairs

### Workshop Flow
1. **Opening**: Context setting, participant introductions
2. **Teaching**: Brief concept introduction (20% of time)
3. **Practice**: Hands-on canvas work (60% of time)
4. **Share**: Peer presentations and feedback (20% of time)

### Post-Workshop Follow-up
1. Canvas submission for review
2. Customer interview assignments
3. Iteration check-in sessions
4. Mentorship pairing
