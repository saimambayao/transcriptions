## Core Competencies

### Seven Essential SE Competencies

| # | Competency | Description | How to Develop |
|---|------------|-------------|----------------|
| 1 | **Leadership** | Take initiative, solve problems proactively | Role-play exercises, project leadership |
| 2 | **Optimism** | Confidence in bold visions despite doubt | Vision articulation, success stories |
| 3 | **Grit** | Perseverance, passion, relentless drive | Challenge-based learning, reflection |
| 4 | **Resilience** | Overcome adversity, learn from failure | Failure analysis, recovery planning |
| 5 | **Creativity** | See new possibilities, unconventional thinking | Design thinking, brainstorming |
| 6 | **Empathy** | Understand beneficiary perspectives | Immersion, beneficiary interviews |
| 7 | **Emotional/Social Intelligence** | Connect with others, build relationships | Team projects, networking exercises |

### Competency Development Principle

**Key insight**: Each competency is learnable with deliberate practice. Treat as skills to develop, not fixed traits.

### Competency-Learning Activity Mapping

| Competency | Effective Activities |
|------------|---------------------|
| Leadership | Lead project teams, facilitate sessions |
| Optimism | Vision boarding, future scenario planning |
| Grit | Long-term projects, obstacle courses |
| Resilience | Post-mortem analysis, pivot exercises |
| Creativity | Ideation workshops, constraints challenges |
| Empathy | Beneficiary shadowing, empathy mapping |
| EQ/SQ | Team dynamics exercises, stakeholder meetings |

## Module Structure

### Standard 8-Module SE Course

#### Module 1: Introduction to Social Entrepreneurship

**Topics**:
- What is social entrepreneurship?
- SE vs traditional business vs NGO
- History and evolution of SE
- SE ecosystem overview
- **BARMM Context**: CSEA mandate, BTA Bill 210, SE landscape in Bangsamoro

**Learning Outcomes**:
- Define social entrepreneurship
- Identify SE characteristics
- Understand the SE landscape in BARMM
- Explain the role of CSEA and the Bangsamoro SE ecosystem

#### Module 2: Identifying Social Problems

**Topics**:
- Problem identification methods
- Root cause analysis
- Beneficiary needs assessment
- Systems thinking
- **BARMM Context**: Key challenges from BDP 2023-2028 (poverty, unemployment, informal sector)

**Learning Outcomes**:
- Analyze social problems systematically
- Identify root causes
- Understand beneficiary perspectives
- Map BARMM-specific social challenges to SE opportunities

#### Module 3: Social Business Model Development

**Topics**:
- Social Business Model Canvas
- Value proposition design
- Revenue model options
- Sustainability planning
- **BARMM Context**: Halal industry opportunities, BIMP-EAGA market access, value chain development

**Learning Outcomes**:
- Complete Social Business Model Canvas
- Design compelling value proposition
- Identify sustainable revenue mix
- Integrate Shari'ah-compliant business practices

#### Module 4: Impact Measurement

**Topics**:
- Theory of Change
- Impact frameworks (IRIS+, SROI)
- Indicator selection
- Data collection methods
- **BARMM Context**: BDP 2023-2028 targets and KPIs

**Learning Outcomes**:
- Develop Theory of Change
- Select appropriate metrics aligned with BDP objectives
- Design measurement approach

#### Module 5: Financial Management for SEs

**Topics**:
- SE financial structures
- Budgeting and forecasting
- Funding sources
- Financial sustainability
- **BARMM Context**: Islamic finance, CSEA programs (SAVE, PESO), cooperative funding

**Learning Outcomes**:
- Create SE budget
- Understand funding landscape including Islamic banking options
- Plan for financial sustainability using Shari'ah-compliant methods

#### Module 6: Marketing and Communication

**Topics**:
- Stakeholder communication
- Impact storytelling
- Brand building
- Digital marketing for SEs
- **BARMM Context**: Halal certification as market differentiator, BIMP-EAGA export potential

**Learning Outcomes**:
- Craft compelling impact stories
- Develop marketing strategy
- Communicate to multiple stakeholders
- Position products for Halal markets

#### Module 7: Legal Structures and Governance

**Topics**:
- Legal entity options
- Governance structures
- Compliance requirements
- BARMM-specific regulations
- **BARMM Context**: BTA Bill 210, CSEA registration, cooperative compliance

**Learning Outcomes**:
- Select appropriate legal structure
- Design governance model
- Understand BARMM compliance requirements (address 11.66% to 50% target)

#### Module 8: Scaling and Sustainability

**Topics**:
- Scaling strategies
- Partnership development
- Sustainability planning
- Exit and transition
- **BARMM Context**: LAKAS SE Program, BIMP-EAGA integration, industry roadmaps

**Learning Outcomes**:
- Develop scaling strategy
- Plan for long-term sustainability
- Build strategic partnerships within BARMM ecosystem

## Learning Levels

### Three-Tier Framework

| Level | Target Audience | Focus | Duration |
|-------|-----------------|-------|----------|
| **Foundational** | New to SE | Concepts, awareness | 8-16 hours |
| **Intermediate** | Early-stage SE leaders | Skills, application | 24-40 hours |
| **Advanced** | Experienced SE practitioners | Mastery, leadership | 40+ hours |

### Foundational Level

**Objectives**:
- Understand SE concepts and frameworks
- Explore SE as a career/business option
- Develop initial SE ideas

**Delivery**:
- Lectures and presentations
- Case study discussions
- Reflection exercises

**BDP 2023-2028 Focus**: Awareness of BDP priorities, understanding SE opportunities in priority sectors

### Intermediate Level

**Objectives**:
- Apply SE frameworks to real challenges
- Develop actionable business plans
- Build practical skills

**Delivery**:
- Workshops and labs
- Mentorship sessions
- Project-based learning

**BDP 2023-2028 Focus**: Developing compliant cooperatives, accessing CSEA programs, value chain participation

### Advanced Level

**Objectives**:
- Lead and scale SE operations
- Navigate complex challenges
- Contribute to SE ecosystem

**Delivery**:
- Peer learning circles
- Executive coaching
- Action research projects

**BDP 2023-2028 Focus**: Cooperative profitability (Strategy 21), policy advocacy (Strategy 22), BIMP-EAGA integration

## Case Study Methodology

### Purpose

Place learners in real-world scenarios to develop:
- Critical thinking
- Decision-making skills
- Stakeholder analysis ability

### Case Study Components

1. **Background** - Organization history, context
2. **Situation** - Current challenge or opportunity
3. **Stakeholders** - Key people and interests
4. **Options** - Possible courses of action
5. **Questions** - Discussion prompts

### Analysis Framework

When analyzing SE cases, explore:

| Element | Questions |
|---------|-----------|
| **Social Mission** | What is the intended impact? |
| **Business Model** | How does it sustain itself? |
| **Stakeholders** | Who are key stakeholders? What are their interests? |
| **Theory of Change** | How does the SE create change? |
| **Challenges** | What obstacles exist? |
| **Decisions** | What choices must be made? |

### Recommended Case Sources

| Source | Focus |
|--------|-------|
| Yale SOM Cases | Global SE cases |
| Harvard Kennedy School | Policy and SE |
| Stanford Social Innovation Review | Thought leadership |
| PhilSEN/British Council | Philippine context |

### BARMM-Specific Case Study Opportunities

Based on BDP 2023-2028, develop local cases around:

| Sector | Case Opportunity | BDP Context |
|--------|------------------|-------------|
| **Seaweed Cooperatives** | Tawi-Tawi seaweed value chain (48.42% national production) | Post-harvest processing, export potential, BIMP-EAGA linkages |
| **Coffee Cooperatives** | Sulu/Maguindanao coffee production (17.4% national share) | Industry Roadmap implementation, value-adding |
| **Halal Enterprise** | Halal certification and market access | Strategic Investment Priority unique to BARMM |
| **Weaving Cooperatives** | Traditional weaving as cultural enterprise | Industry Roadmap for Weaving, women empowerment |
| **Fisher Cooperatives** | Sustainable fishing and aquaculture | 252,758 registered fisherfolk, highest in country |
| **Former Combatant Coops** | Transition from conflict to enterprise | Combatant Transitional Program, compliance challenges |
| **Barter Trade SEs** | Cross-border trade with BIMP-EAGA | Greater Sulu-Sulawesi Corridor, cultural trading practice |
| **Cassava Processing** | Value chain for 41.70% national production | Post-harvest facilities, Halal processing |

**Case Development Focus Areas (from BDP 2023-2028 challenges):**
- Cooperative compliance improvement (from 11.66% to 50% target)
- MSME formalization (informal sector dominance - only 0.92% of national BNR)
- Shari'ah-compliant business models
- Value chain development for local commodities
- Women-led enterprise development (85% participation in SUBATRA-ECSO)
- Former combatant cooperative transition
- LGU-cooperative coordination (MCO establishment)

### Case Facilitation Tips

1. **Prepare thoroughly** - Know the case deeply
2. **Start with facts** - Establish shared understanding
3. **Encourage multiple perspectives** - No single right answer
4. **Connect to theory** - Link to frameworks
5. **Apply to context** - Relate to learners' situations
6. **Reference BDP 2023-2028** - Ground discussions in regional priorities

**Components**:
- Self-assessment of competencies
- Learning reflections
- Project documentation
- Growth evidence

**Timeline**: Developed throughout course (5+ weeks)

**Components**:

| Section | Weight |
|---------|--------|
| Social Mission & Theory of Change | 20% |
| Social Business Model Canvas | 20% |
| Market Analysis | 15% |
| Financial Projections | 15% |

| Implementation Plan | 15% |

**BARMM-Specific Criteria**:
- Alignment with BDP 2023-2028 sectoral objectives
- Shari'ah compliance considerations
- Halal market opportunity identification
- Cooperative compliance pathway

### Impact Measurement Exercise

**Task**: Design impact measurement approach for a case SE

**Criteria**:
- Theory of Change clarity
- Indicator appropriateness (aligned with BDP KPIs)
- Data collection feasibility
- Analysis plan

### Community Project

**Structure**:
- Team-based (3-5 members)
- Real community partner
- Defined deliverable
- Presentation to stakeholders

**Assessment Criteria**:
- Community impact achieved
- Team collaboration
- Learning demonstrated
- Reflection quality
- BDP 2023-2028 alignment

**Format**:
- 5-minute pitch
- 3-minute Q&A
- Panel of SE practitioners/investors

**Criteria**:
- Clarity of social mission
- Business model viability
- Impact potential
- Presentation quality
- BARMM market relevance

# General -> BARMM Contextualization Standard

#### General → BARMM Contextualization Standard

**CRITICAL**: This pedagogical standard MUST be applied to ALL social enterprise curriculum design.

##### The Two-Layer Approach

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│  LAYER 1: GENERAL/GLOBAL CONTEXT (Always First)                            │
│  ┌────────────────────────────────────────────────────────────────────────┐ │
│  │ Present what is taught GENERALLY worldwide and nationwide:             │ │
│  │   • EU/OECD social enterprise definitions                              │ │
│  │   • B-Corp certification standards                                     │ │
│  │   • IRIS+ impact measurement framework                                 │ │
│  │   • Social Business Model Canvas                                       │ │
│  │   • Theory of Change methodology                                       │ │
│  │   • Philippine SE context (PRESENT Bill, RA 11524)                     │ │
│  │   • Standard SE governance and legal structures                        │ │
│  └────────────────────────────────────────────────────────────────────────┘ │
│                                    ↓                                        │
│  LAYER 2: BARMM CONTEXTUALIZATION (Then Second)                            │
│  ┌────────────────────────────────────────────────────────────────────────┐ │
│  │ Present BARMM-specific application and contextualization:              │ │
│  │   • CSEA dual mandate (cooperatives + social enterprises)              │ │
│  │   • BTA Bill 210 SE provisions                                         │ │
│  │   • Islamic social enterprise principles                               │ │
│  │   • Shari'ah-compliant business models                                 │ │
│  │   • Moral Governance 12 pillars in SE context                          │ │
│  │   • BDP 2023-2028 SE development priorities                            │ │
│  │   • Halal value chain requirements                                     │ │
│  └────────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────┘
```

##### Exception: BARMM-First Delivery

**When general context is not applicable or contradictory, BARMM context may be delivered first:**

| Scenario | Reason | Example |
|----------|--------|---------|
| **No global equivalent** | Topic is BARMM-specific with no general counterpart | CSEA-specific SE registration |
| **Contradictory principles** | Global practice conflicts with BARMM requirements | Interest-based financing vs Shari'ah-compliant |
| **Regional-only programs** | Programs that only exist in BARMM | BTA Bill 210 SE provisions |
| **Cultural specificity** | Content inherently tied to Bangsamoro culture | Islamic social enterprise models |

##### Application to SE Topics

| Topic | Layer 1 (General) | Layer 2 (BARMM) |
|-------|-------------------|-----------------|
| **SE Definition** | EU/OECD 3-dimension definition | BTA Bill 210 definition, Islamic social enterprise |
| **Impact Measurement** | IRIS+, SROI, B Impact Assessment | Maqasid al-Shariah alignment, community impact |
| **Business Models** | Social Business Model Canvas | Shari'ah-compliant models, Waqf, Sadaqah integration |
| **Governance** | B-Corp governance standards | Shura principles, Moral Governance integration |
| **Legal Structure** | Corporation, Foundation, BMBE | CSEA registration, BTA Bill 210 structure |
| **Financing** | Impact investing, grants, earned revenue | Murabaha, Musharakah, Islamic microfinance |

##### Decision Tree

```text
START: Design SE curriculum content
    │
    ├─► Does a GENERAL/GLOBAL standard exist for this topic?
    │   │
    │   ├─► YES: Does it conflict with BARMM/Islamic principles?
    │   │   │
    │   │   ├─► NO CONFLICT: Use Two-Layer Approach
    │   │   │   Layer 1: Present global/national standard first
    │   │   │   Layer 2: Show BARMM contextualization
    │   │   │
    │   │   └─► CONFLICT: Use BARMM-First
    │   │       Present BARMM-compliant alternative
    │   │       Explain why global practice differs
    │   │
    │   └─► NO: Use BARMM-First
    │       Present BARMM-specific content directly
    │       Note that this is regional-specific
    │
    └─► END
```

##### Implementation Checklist

When designing SE curriculum:

- [ ] **Identify global standards first**: What do EU/OECD, B-Corp, GIIN teach about this topic?
- [ ] **Check for conflicts**: Does the global standard conflict with Islamic/BARMM principles?
- [ ] **Structure two layers**: If no conflict, present global THEN BARMM context
- [ ] **Note exceptions**: If conflict exists, explain the BARMM-compliant alternative
- [ ] **Cross-reference sources**: Cite both global frameworks AND CSEA/BTA sources
- [ ] **Validate with /academy**: Ensure curriculum follows MoroAcademy standards
