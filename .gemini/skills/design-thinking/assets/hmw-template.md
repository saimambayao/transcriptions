## HMW Quality Checklist

Rate your HMW statement:

| Criterion | Check | Notes |
|-----------|-------|-------|
| **Grounded in research** - Based on real user insights, not assumptions | ☐ | |
| **Solution-agnostic** - Doesn't embed a specific solution | ☐ | |
| **Right breadth** - Not too narrow, not too broad | ☐ | |
| **Outcome-focused** - Addresses root cause, not symptoms | ☐ | |
| **Positive framing** - Action-oriented, possibility language | ☐ | |
| **Inspires ideas** - Opens up creative possibilities | ☐ | |

## Common HMW Mistakes (and Fixes)

### Mistake 1: Embedded Solution
❌ "How might we build an app for cooperative members?"
✅ "How might we help cooperative members access information easily?"

### Mistake 2: Too Broad
❌ "How might we improve the world?"
✅ "How might we help farmers access fair prices for their produce?"

### Mistake 3: Too Narrow
❌ "How might we add a blue button to the registration page?"
✅ "How might we make registration feel simple and welcoming?"

### Mistake 4: Not Grounded in Research
❌ "How might we make users happier?" (generic)
✅ "How might we reduce the anxiety Maria feels when submitting compliance documents?"

### Mistake 5: Symptom-Focused
❌ "How might we stop members from calling customer support?"
✅ "How might we help members feel confident they have the information they need?"

```

### Angle 3: Explore the Opposite

```

```

### Angle 4: Question Assumptions

```
How might we _________________________________________________?
```

```

## Examples for BARMM Context

### Cooperative Example

**Insight**: Coop officers spend 60% of their time on manual record-keeping instead of member services.

**HMW Variations**:
1. "How might we automate routine record-keeping for coop officers so they can focus on member support?"
2. "How might we make compliance reporting feel effortless for small cooperatives?"
3. "How might we help coop officers see record-keeping as valuable rather than burdensome?"

### Social Enterprise Example

**Insight**: SE founders struggle to explain their social mission to potential investors.

**HMW Variations**:
1. "How might we help SE founders communicate impact in ways investors understand?"
2. "How might we create tools that make social impact visible and compelling?"
3. "How might we connect SE founders with investors who value social return?"

### Government Service Example

**Insight**: Citizens don't know which CSEA services are available to them.

**HMW Variations**:
1. "How might we make CSEA services discoverable at the moment citizens need them?"
2. "How might we help communities spread awareness of CSEA programs?"
3. "How might we meet citizens where they already gather information?"

## Team Workshop Process

1. **Individual HMW Writing** (5 min)
   - Each person writes 3-5 HMWs on sticky notes
   - One HMW per sticky note

2. **Share and Cluster** (10 min)
   - Post all HMWs on wall
   - Group similar HMWs
   - Identify themes

3. **Dot Voting** (5 min)
   - Each person gets 3 votes
   - Vote on most promising HMWs

4. **Selection** (5 min)
   - Choose top 1-3 HMWs for ideation
   - Refine wording if needed
