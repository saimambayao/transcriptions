# EDP Integration for Product Management

Practical frameworks from MoroAcademy's Enterprise Development Program, contextualized for BARMM cooperatives and social enterprises.

## The 3 Big Questions (Module 3)

Every product must answer:

| Question | Focus | Output |
|----------|-------|--------|
| **WHAT** | Clear product description | Product specification |
| **HOW** | Step-by-step process | Production workflow |
| **HOW MUCH** | Cost per item & profit | Unit economics |

---

## Product Description Elements

| Element | Question to Answer | Example |
|---------|-------------------|---------|
| **Product Name** | What is it called? | Chicken Pastil |
| **What It Is** | Description (1-2 sentences) | Steamed rice, savory kagikit, banana leaf wrap |
| **Size/Weight** | How much per unit? | Snack-sized, 3 pieces = 1 meal |
| **Packaging** | Container & label | Banana leaf + branded sticker |
| **Features** | What makes it good? | Fresh daily, Halal, authentic recipe |
| **Shelf Life** | How long does it last? | 24 hours (fresh) |
| **Target Customer** | Who will buy it? | Government workers, busy families |

### Good vs Bad Descriptions

| Bad (Vague) | Good (Clear) |
|-------------|--------------|
| "Pastil" | "Chicken Pastil - 150g steamed rice, savory shredded chicken (kagikit), wrapped in banana leaf. Halal." |
| "Vegetables" | "Organic salad mix - 200g assorted lettuce, tomatoes, cucumber. Washed and ready to eat. Best within 3 days." |

## Unit Cost Formula

```
Cost Per Item = Bulk Price / Quantity Made
```

### Example: Pastil Cost Sheet

| Material | Bulk Price | Yields | Cost/Piece |
|----------|-----------|--------|------------|
| Rice (25kg) | P1,000 | 360 pcs | P2.78 |
| Chicken (1kg) | P180 | 60 pcs | P3.00 |
| Banana Leaves | P50 | 100 pcs | P0.50 |
| Oil/Spices | P220 | - | P1.60 |
| LPG/Labor | P1,000 | - | P2.10 |
| **TOTAL** | | | **P9.98** |

### Profit Calculation

```
Selling Price:  P15.00
- Total Cost:   P 9.98
= PROFIT:       P 5.02  (33% margin)
```

### Variable vs Fixed Costs

| Type | Definition | Examples |
|------|------------|----------|
| **Variable** | Changes with volume | Rice, chicken, leaves (make more = buy more) |
| **Fixed** | Stays same regardless | Stall rent, equipment |

## MVP Framework (Module 3)

### Why MVP?

**Common failures:**
- Invest everything BEFORE testing
- Assume customers want it
- Build full-scale too early

**Real story**: A cooperative invested P150,000 in a facility. After building it, they found customers preferred fresh products. The facility now sits unused.

### Full Launch vs MVP

| Approach | Investment | Production | Risk |
|----------|-----------|------------|------|
| **Full Launch** | P500,000 (Equipment) | 1,000 units (Hope) | HIGH (Gamble) |
| **MVP Way** | P5,000 (Test) | 50 units (Know) | LOW (Smart) |

### MVP Test Template

| Element | Plan Example |
|---------|--------------|
| **Quantity** | 50 pcs x 3 days = 150 Total |
| **Duration** | 3 Days (Fri-Sun) |
| **Location** | BGC / Target market |
| **Target** | Government Workers |
| **Price** | P15.00 |
| **Budget** | P1,100 |

### What to Test

| Question | How to Test |
|----------|-------------|
| Will they buy? | Offer for sale, track purchases |
| Is price right? | Test different prices |
| Is quality ok? | Get customer feedback |
| Can we deliver? | Test delivery to few customers |

### Success Metrics

| Metric | Example Target |
|--------|---------------|
| **Sales Target** | 80% (120/150 units) |
| **Feedback** | 90% positive |
| **Repeats** | 5+ return customers |
| **Break-even** | 74 units to cover P1,100 |

### Coop Advantage: Pre-Orders

At the coop meeting, announce your test. Members can pre-order:
- Guaranteed sales BEFORE cooking
- Honest feedback from members
- Borrowed equipment from Service Coop

## Decision Framework

### Scale Up / Fix / Adjust / Stop

| Result | Meaning | Action |
|--------|---------|--------|
| High sales + Good feedback | Product-Market Fit | **SCALE UP** |
| High sales + Complaints | Quality Issues | **FIX & RETEST** |
| Low sales + Good feedback | Wrong Price/Place | **ADJUST** |
| Low sales + No interest | Wrong Product | **STOP / PIVOT** |

### Decision Tree

```
Did you sell 80%?
├── YES → Feedback Positive?
│         ├── YES → SCALE UP!
│         └── NO → FIX & RETEST
│
└── NO → Was there interest?
         ├── YES → ADJUST (price/place)
         └── NO → STOP / PIVOT
```

## Value Chain (Module 5)

### The 5 Stages

```
INPUT → PRODUCTION → PROCESSING → DISTRIBUTION → CONSUMER
(Supplies)  (Growing)    (Value Add)   (Selling)     (End User)
```

| Stage | Activities | Coop Example |
|-------|-----------|--------------|
| **Input** | Seeds, tools, supplies | Agricultural Supply Coop |
| **Production** | Farming, fishing, harvesting | Farmers/Fishermen Coop |
| **Processing** | Cooking, milling, packaging | Women's Producers Coop |
| **Distribution** | Market stalls, delivery | Consumer Cooperative |
| **Consumer** | Households, restaurants | End buyers |

### 5-Cooperative Network

| Coop Type | Stage | Role | Value Added |
|-----------|-------|------|-------------|
| Agri Supply | INPUT | Seeds/Tools | Quality Input |
| Farmers | PRODUCTION | Grow Rice | Raw Material |
| Fishermen | PRODUCTION | Catch Fish | Protein Source |
| Producers | PROCESSING | Make Food | Convenience/Taste |
| Consumer | DISTRIBUTION | Retail Store | Accessibility |

### Service Coop Role

Service Cooperatives support ALL stages:
- Utilities
- Transport
- Logistics
- Finance (Murabaha)

## Financial Calculations

### Break-Even Analysis

```
Break-Even Units = Fixed Costs / (Price - Variable Cost)
```

**Example:**
- Average Price: P28
- Variable Cost: P12
- Margin per Unit: P16
- Fixed Costs: P37,000/month
- **Break-Even: 2,313 pcs/month (43% of target)**

### Payback Period

If using ALL net profit to repay loan:
- Loan: P85,000
- Net Profit: P33,708/month
- **Payback: ~3 months**
