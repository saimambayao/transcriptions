## Value Chain Development Priorities

### Agriculture Value Chain

```
PRODUCTION -> POST-HARVEST -> PROCESSING -> MARKETING -> DISTRIBUTION
    |              |              |             |              |
    v              v              v             v              v
Farm inputs    Storage &     Value-adding   Market        Barter trade
Mechanization  Drying        Facilities     linkages      BIMP-EAGA
Irrigation     FMRs          Canning        Negosyo       Export
Extension      Cold storage  Halal cert     Centers       Local retail
```

**Priority Interventions:**
1. Pre-harvest: Farm inputs, mechanization, irrigation
2. Post-harvest: Storage, drying, cold chain facilities

## Social Enterprise Adaptations

### Dual-Purpose Product Design

Every feature should address:
1. **Impact delivery** - How does it serve beneficiaries?
2. **Sustainability** - How does it generate/save revenue?

### Impact-First Prioritization

```
PRIORITIZATION FILTER:
|
+-- Does it advance our mission?
|   +-- NO -> Don't build (usually)
|   +-- YES |
|           v
+-- Does it generate/save money?
|   +-- YES -> High priority
|   +-- NO |
|          v
+-- Is it required for operations?
    +-- YES -> Medium priority
    +-- NO -> Low priority (validate further)
```

### SE Product Categories

| Category | Examples | Priority Lens |
|----------|----------|---------------|
| **Impact Products** | Programs, services for beneficiaries | Mission-first |
| **Revenue Products** | Paid services, products | Sustainability-first |
| **Hybrid Products** | Cross-subsidized offerings | Balance both |
| **Operational** | Internal tools | Efficiency |
