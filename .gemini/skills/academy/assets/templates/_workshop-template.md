## Facilitator Guide

### Before the Workshop

**2 Weeks Before**:
- [ ] Confirm venue and logistics
- [ ] Send participant confirmation
- [ ] Prepare all materials
- [ ] Brief co-facilitators

**1 Day Before**:
- [ ] Set up venue
- [ ] Test all equipment
- [ ] Arrange seating
- [ ] Prepare registration area

**Day of Workshop**:
- [ ] Arrive 30-60 min early
- [ ] Final equipment check
- [ ] Review agenda and timing
- [ ] Brief support staff

### During the Workshop

**Time Management**:
- Use visible timer
- Provide 5-minute warnings
- Have buffer activities ready
- Stick to breaks (participants need them)

**Engagement Strategies**:
- Change activity type every 20-30 min
- Use energizers after lunch
- Encourage cross-table interaction
- Celebrate small wins

**Troubleshooting**:

| Issue | Solution |
|-------|----------|
| Dominant participant | Direct questions to others, use structured turn-taking |
| Silent group | Use pair-share before group discussion |
| Running behind | Cut discussion time, not activity time |
| Technical failure | Have offline backup materials ready |
| Conflict | Use mediation techniques, take break if needed |

### After the Workshop

**Same Day**:
- [ ] Collect all outputs and assessments
- [ ] Back up photos and materials
- [ ] Note improvements for next time

**Within 1 Week**:
- [ ] Send thank you and resources to participants
- [ ] Process certificates
- [ ] Complete workshop report

**Follow-up**:
- [ ] Week 2: Check-in call/message
- [ ] Month 1: Progress review
- [ ] Month 3: 90-day milestone (if applicable)
