#### Lesson 1.2: Officers vs. Management Responsibilities (VIDEO - 7 min)

**Opening Hook** (30 sec):
"Fatima asked a great question - how is being a director different from being a manager? This distinction is crucial for effective governance."

**Core Content** (6 min):

**SCENARIO**: Fatima's Transition

> Fatima has been MoroCoop's store manager for 5 years. Now elected to the BOD, she attends her first board meeting. The agenda includes approving the annual budget. She starts reviewing every line item, questioning small expenses.
>
> Hajji Ibrahim gently interrupts: "Fatima, as a director, your job is different now. You approve the budget framework, not every purchase order."

**The Governance vs. Management Distinction:**

```
+---------------------------+---------------------------+
|       GOVERNANCE          |       MANAGEMENT          |
|       (Officers/BOD)      |       (Staff/Manager)     |
+---------------------------+---------------------------+
|                           |                           |
| WHAT & WHY                | HOW                       |
| (Strategic Direction)     | (Implementation)          |
|                           |                           |
+---------------------------+---------------------------+
|                           |                           |
| Approve policies          | Implement policies        |
| Set goals & targets       | Execute plans             |
| Monitor results           | Produce results           |
| Hire/fire GM              | Supervise employees       |
| Approve budget            | Manage expenses           |
|                           |                           |
+---------------------------+---------------------------+
|                           |                           |
| Ask: "Are we doing        | Ask: "Are we doing        |
| the RIGHT things?"        | things RIGHT?"            |
|                           |                           |
+---------------------------+---------------------------+
```

**Common Role Confusion Problems:**

| Problem | Description | Solution |
|---------|-------------|----------|
| **Micro-management** | Board controls daily operations | Focus on policies, not transactions |
| **Rubber-stamping** | Board approves without questioning | Ask probing questions, require information |
| **Role reversal** | Manager makes strategic decisions | Board sets direction, manager executes |
| **Dual roles** | Officer is also employee | Clear separation, declare conflicts |

**Bridge** (30 sec):
"Understanding this distinction will help you be effective in your specific role. Next, let's look at how accountability translates to practical obligations."

#### Lesson 1.3: Practical Accountability Framework (VIDEO - 7 min)

**Opening Hook** (30 sec):
"Now let's build a practical framework you can use to navigate accountability in your daily officer duties."

**Core Content** (6 min):

**The 4-Question Accountability Test:**

Before any decision or action, officers should ask:

```
ACCOUNTABILITY TEST
+-----------------------------------------------+
|                                               |
|  1. AUTHORITY: Do I have the authority to     |
|     take this action?                         |
|     - Is it within my role?                   |
|     - Has it been delegated to me?            |
|     - Does it require board approval?         |
|                                               |
|  2. PROCESS: Am I following proper process?   |
|     - Is there a policy for this?             |
|     - Do I have required documentation?       |
|     - Are proper approvals in place?          |
|                                               |
|  3. BENEFIT: Is this in the cooperative's     |
|     best interest?                            |
|     - Does it benefit members?                |
|     - Is there a conflict of interest?        |
|     - Would a prudent person do this?         |
|                                               |
|  4. DOCUMENTATION: Can I document and         |
|     justify this decision?                    |
|     - Is there a record?                      |
|     - Can I explain it to auditors?           |
|     - Would I be comfortable if this          |
|       appeared in a CDA inspection?           |
|                                               |
+-----------------------------------------------+
```

**SCENARIO**: Mang Karding's Disbursement Decision

> A member urgently needs their withdrawal of P50,000 share capital. The member says it's a medical emergency. Normal process requires board approval for withdrawals over P30,000, but the next board meeting is in 2 weeks.
>
> The manager asks Mang Karding to approve it as treasurer.

**Applying the 4-Question Test:**

| Question | Analysis |
|----------|----------|
| **Authority** | Treasurer can approve up to P30K. This exceeds limit. |
| **Process** | Policy requires board approval for >P30K |
| **Benefit** | Helping member vs. setting precedent |
| **Documentation** | How would this look to auditors? |

**Correct Action**: Emergency board meeting or canvassing of directors for approval, not unilateral action.

**Bridge** (30 sec):
"This framework applies to every officer role. In the next lesson, we'll explore the specific types of situations that create legal exposure."

#### Lesson 2.1: The Three Fiduciary Duties (VIDEO - 7 min)

**Opening Hook** (30 sec):
"As cooperative officers, you are fiduciaries - people entrusted with the care of others' interests. This creates three fundamental duties that guide everything you do."

**Core Content** (6 min):

**The Fiduciary Triad:**

```
              DUTY OF CARE
                  /\
                 /  \
                /    \
               /      \
              / OFFICER \
             /  DUTIES   \
            +-----------+
           /              \
          /                \
    DUTY OF              DUTY OF
    LOYALTY              OBEDIENCE
```

**1. Duty of Care:**

> Act with the care that a reasonably prudent person would exercise in similar circumstances.

| Requirement | Meaning | Example |
|-------------|---------|---------|
| **Stay Informed** | Know what's happening | Read reports before meetings |
| **Attend Meetings** | Be present and engaged | Don't miss board meetings |
| **Ask Questions** | Seek clarification | Don't approve what you don't understand |
| **Make Prudent Decisions** | Be reasonable | Consider risks and alternatives |

**2. Duty of Loyalty:**

> Put the cooperative's interests above your own personal interests.

| Requirement | Meaning | Example |
|-------------|---------|---------|
| **No Self-Dealing** | Don't benefit personally from position | Don't award contracts to your business |
| **Confidentiality** | Protect sensitive information | Don't share member data |
| **Avoid Conflicts** | Declare and manage conflicts | Recuse yourself from related decisions |
| **Good Faith** | Act honestly and fairly | Don't deceive other officers |

**3. Duty of Obedience:**

> Ensure the cooperative operates within its stated purpose and legal requirements.

| Requirement | Meaning | Example |
|-------------|---------|---------|
| **Follow Laws** | Comply with RA 9520, CDA rules | File required reports |
| **Follow Bylaws** | Operate per cooperative's bylaws | Conduct meetings properly |
| **Pursue Mission** | Stay true to cooperative purpose | Don't drift from stated objectives |
| **Member Interest** | Serve member welfare | Decisions benefit membership |

**Bridge** (30 sec):
"These duties aren't just theoretical - they come up constantly. Let's see how to handle conflicts of interest practically."

#### Lesson 2.2: Conflict of Interest Management (VIDEO - 7 min)

**Opening Hook** (30 sec):
"Let's return to Danny's brother-in-law situation. Even good intentions can create conflicts that must be properly managed."

**Core Content** (6 min):

**Types of Conflicts of Interest:**

```
CONFLICT OF INTEREST TYPES
+--------------------------------------------------+
|                                                  |
|  DIRECT: You personally benefit                  |
|  - Your business gets a contract                 |
|  - You receive a commission                      |
|                                                  |
|  INDIRECT: Related party benefits                |
|  - Your family member's business                 |
|  - Your close friend's company                   |
|                                                  |
|  APPARENT: Looks like a conflict                 |
|  - Even if there's no actual benefit             |
|  - Perception matters                            |
|                                                  |
|  POTENTIAL: Could become a conflict              |
|  - You're negotiating employment                 |
|  - Future benefit possibility                    |
|                                                  |
+--------------------------------------------------+
```

**The Conflict Management Process:**

```
Step 1: IDENTIFY
   Is there any potential conflict?
            |
            v
Step 2: DISCLOSE
   Report the conflict to the board
            |
            v
Step 3: RECUSE
   Remove yourself from discussion and vote
            |
            v
Step 4: DOCUMENT
   Record disclosure and recusal in minutes
            |
            v
Step 5: REVIEW
   Remaining board decides without you
```

**Back to Danny's Situation:**

Danny should:
1. **Disclose**: "I have a conflict - the trucking company is owned by my brother-in-law"
2. **Recuse**: Leave the room during discussion and voting
3. **Accept Decision**: Whether approved or not, Danny must accept the board's decision

**Even if approved**, the board must:
- Get competitive quotes from other trucking companies
- Document that Danny's brother-in-law's terms are truly competitive
- Record the conflict disclosure in the minutes

#### Lesson 2.3: Ethical Decision-Making Framework (VIDEO - 7 min)

**Opening Hook** (30 sec):
"Beyond managing conflicts, officers face ethical dilemmas that don't have clear right answers. Here's a framework to navigate these situations."

**Core Content** (6 min):

**The ETHICS Framework:**

```
E - Evaluate the situation factually
T - Think about stakeholders affected
H - Hear different perspectives
I - Identify options and consequences
C - Choose based on values and duties
S - Stand accountable for your decision
```

**SCENARIO**: The Struggling Member

> A long-time member, Hadja Salma, has a P100,000 loan that's 90 days past due. Her husband recently died, and she's struggling. The Credit Committee must decide: pursue collection aggressively, restructure the loan, or write it off.

**Applying the ETHICS Framework:**

| Step | Application |
|------|-------------|
| **E - Evaluate** | P100K loan, 90 days PDO, member for 15 years, good history before tragedy |
| **T - Think** | Hadja Salma, her children, other borrowers (precedent), cooperative's finances |
| **H - Hear** | Talk to Hadja Salma, get financial assessment, consult policy |
| **I - Identify** | Options: aggressive collection, restructure, partial write-off, community support fund |
| **C - Choose** | Based on 7th Principle (Concern for Community) + financial prudence |
| **S - Stand** | Document reasoning, be prepared to explain decision |

**Islamic Values in Cooperative Ethics** (BARMM Context):

| Islamic Concept | Application |
|-----------------|-------------|
| **Amanah** (Trust) | Officers are trustees of members' resources |
| **Adalah** (Justice) | Fair treatment of all members |
| **Ihsan** (Excellence) | Going beyond minimum requirements |
| **Shura** (Consultation) | Collective decision-making |

#### Lesson 3.1: Officer Coordination Framework (VIDEO - 7 min)

**Opening Hook** (30 sec):
"With 4 business lines and multiple committees, MoroCoop can't afford communication breakdowns. Let's build a coordination framework that keeps everyone aligned."

**Core Content** (6 min):

**Coordination Architecture:**

```
COORDINATION STRUCTURE
                    +------------------+
                    |      BOD         |
                    | (Monthly Meeting)|
                    +--------+---------+
                             |
      +----------------------+----------------------+
      |                      |                      |
+-----v-----+         +------v------+        +-----v-----+
| Executive |         |  Committee  |        | Management|
|  Committee|         |Coordination |        | Interface |
| (Weekly)  |         |  (Monthly)  |        | (Weekly)  |
+-----------+         +-------------+        +-----------+
      |                      |                      |
      v                      v                      v
  Chair +              Audit + Credit          GM reports
  Treasurer +          + Election +            to BOD
  Secretary            E&T Committees
```

**Types of Coordination Meetings:**

| Meeting | Frequency | Participants | Purpose |
|---------|-----------|--------------|---------|
| **BOD Meeting** | Monthly | All directors | Strategic decisions, approvals |
| **Executive Committee** | Weekly | Chair, Treasurer, Secretary | Urgent matters, coordination |
| **Committee Coordination** | Monthly | All committee chairs | Cross-committee alignment |
| **BOD-Management** | Weekly | Chair + GM | Operations oversight |
| **GA** | Annual | All members | Major decisions, elections |

**Danny & Mang Karding's Solution:**

The Credit Committee and Treasurer should:
1. **Weekly Cash Position Report**: Treasurer shares available funds
2. **Pre-Approval Check**: Credit Committee verifies fund availability before large approvals
3. **Approval Notification**: Credit Committee sends approval list to Treasurer within 24 hours
4. **Monthly Reconciliation**: Joint review of loan disbursements vs. cash position

#### Lesson 3.2: Board-Management Relations (VIDEO - 7 min)

**Opening Hook** (30 sec):
"The relationship between the Board and Management is one of the most critical - and most often mismanaged - in cooperatives. Let's get it right."

**Core Content** (6 min):

**SCENARIO**: The Hiring Disagreement

> MoroCoop's consumer store needs a new manager. The General Manager (GM) identifies a candidate and wants to hire immediately.
>
> Fatima (BOD): "Wait, shouldn't the Board approve this hire?"
>
> GM: "I have authority over staffing. The Board shouldn't micromanage."
>
> Hajji Ibrahim: "Both of you are partially right. Let's clarify."

**Clear Delegation Framework:**

```
DECISION AUTHORITY MATRIX
+--------------------+-------+-------+-------+
| Decision Type      | GM    | Chair | BOD   |
+--------------------+-------+-------+-------+
| Daily operations   | DECIDE| Inform| --    |
| Staff hiring <50K  | DECIDE| Inform| --    |
| Staff hiring >50K  | Propose| Review| DECIDE|
| Contracts <100K    | DECIDE| Inform| --    |
| Contracts 100-500K | Propose| Review| DECIDE|
| Contracts >500K    | Propose| --    | DECIDE|
| Emergency <50K     | DECIDE| Inform| Report|
| Policy changes     | Propose| --    | DECIDE|
| Budget execution   | MANAGE| Monitor| Review|
| Strategic direction| Input | --    | DECIDE|
+--------------------+-------+-------+-------+

DECIDE = Has authority to make final decision
Propose = Recommends to decision-maker
Review = Examines before decision
Inform = Receives notification after decision
Monitor = Ongoing oversight
Report = After-the-fact report to BOD
```

**The One-Boss Principle:**

The GM reports to the **Board as a whole**, not to individual directors.

| Wrong | Right |
|-------|-------|
| Director directly instructing GM | Board resolution directing GM |
| Chair making decisions alone | Chair facilitating board decisions |
| Director bypassing GM to staff | Director raising concerns at board meeting |

   - Date, time, place
   - Presiding officer
   - Directors present/absent (names)
   - Quorum declaration

2. APPROVAL OF PREVIOUS MINUTES
   - Motion, second, vote

3. AGENDA ITEMS (for each):
   - Item description
   - Discussion summary (key points, not verbatim)
   - Motion (exact wording)
   - Seconded by
   - Vote (favor/against/abstain)
   - Resolution number

4. OTHER MATTERS

5. ADJOURNMENT
   - Time adjourned
   - Next meeting date

SIGNATURES:
- Secretary (preparer)
- Chairperson (attester)
