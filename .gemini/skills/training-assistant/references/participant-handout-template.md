# Participant Handout Template Structure

This template defines the standard structure for participant handouts/workbooks generated from CSEA-BARMM training designs. Participant handouts serve as the learner's companion throughout the training and as a take-home reference afterward.

---

## Document Styling

- **Font**: Inter throughout
- **Body**: 10pt, line spacing 1.0, single extra line between paragraphs
- **Headings**: H1 = 16pt bold `#1B4F72`, H2 = 13pt bold `#1B4F72`, H3 = 11pt bold `#2E86C1`
- **Accent color**: `#1B4F72` for section dividers, key term boxes, and header bars
- **Key term boxes**: Light blue-gray background `#D6EAF8` with left border `#1B4F72`
- **Notes areas**: Light gray background `#F2F3F4` with dotted border
- **Table headers**: `#1B4F72` background, white text
- **Checkbox style**: Use Unicode ballot box characters
- **Reading level**: Target Grade 6-8 (accessible for BARMM cooperative members where functional literacy is ~59.3%)
- **No purple colors**
- **Icons only, no emojis**

### TOPIC SECTIONS

For EACH topic in the training design, create a section with these components:

#### Topic {N}: {Topic Title}

**A. Key Concepts**

Write 1-3 short paragraphs summarizing the essential ideas for this topic. Use plain language. Define technical terms inline.

Example format:
> A cooperative is a group of people who come together voluntarily to solve a shared problem or meet a common need. Unlike a regular business, a cooperative is owned and controlled by its members -- the same people it serves.

**B. Key Terms**

Present in a styled box:

```
+-------------------------------------------+
| KEY TERMS                                  |
|-------------------------------------------|
| Cooperative -- A group of people who...    |
| Articles of Cooperation -- The legal...    |
| Bylaws -- The internal rules that...       |
+-------------------------------------------+
```

Use 5-8 terms per topic maximum. Keep definitions to one sentence.

**C. Activity Worksheet**

Design worksheets that match the activities described in the training design session plan. Worksheet types:

1. **Fill-in-the-blank / Completion**
   Good for: definitions, principles, key facts
   ```
   The ______ Cooperative Principles were established by the ICA.
   The first principle is: ________________________________
   ```

2. **Matching Exercise**
   Good for: connecting concepts to definitions, examples to categories
   ```
   Match the business type with its ownership structure:
   ___ Cooperative          A. One owner, all liability
   ___ Corporation           B. Members own equally
   ___ Sole Proprietorship   C. Shareholders own by shares
   ```

3. **Table Completion**
   Good for: comparative analysis, structured outputs
   ```
   | Feature | Cooperative | Corporation | Sole Proprietorship |
   |---------|------------|-------------|---------------------|
   | Who owns it? | | | |
   | Who decides? | | | |
   | How are profits shared? | | | |
   ```

4. **Guided Worksheet**
   Good for: workshops producing specific outputs (e.g., MVO formulation, Theory of Change)
   ```
   OUR COOPERATIVE'S MISSION, VISION, AND OBJECTIVES

   Step 1: What is our cooperative's main purpose?
   _______________________________________________

   Step 2: What values guide our work? (Choose 3 from the list)
   [ ] Self-help        [ ] Equality        [ ] Solidarity
   [ ] Self-responsibility  [ ] Equity      [ ] Honesty
   [ ] Democracy        [ ] Social responsibility
   [ ] Openness         [ ] Caring for others

   Step 3: Write your Mission Statement:
   _______________________________________________
   _______________________________________________
   _______________________________________________
   ```

5. **Reflection Prompt**
   Good for: personal connection, end-of-topic synthesis
   ```
   REFLECT:
   How does what you learned today connect to your own cooperative experience?
   _______________________________________________
   _______________________________________________
   _______________________________________________
   ```

6. **Checklist / Self-Assessment**
   Good for: skill verification, action planning
   ```
   After this topic, I can:
   [ ] Explain why cooperatives are formed
   [ ] Name at least 3 cooperative principles
   [ ] Describe how a cooperative is different from a corporation
   ```

**D. Notes Space**

After each topic section, include a boxed area for notes:

```
+-------------------------------------------+

|                                            |
| __________________________________________ |
| __________________________________________ |
| __________________________________________ |
| __________________________________________ |
| __________________________________________ |
| __________________________________________ |
+-------------------------------------------+
```

Provide 8-12 lines of notes space per topic.

### REFERENCE SHEETS

After all topic sections, include reference materials that participants can use during and after the training:

**Types of reference sheets to include (based on training content):**

1. **Framework Summary**
   One-page visual summary of key frameworks used in the training (e.g., Kolb Cycle, Theory of Change model, 5-Criteria Comparison Framework)

2. **Quick Reference Table**
   Consolidated version of key facts/data covered (e.g., 7 Cooperative Principles with one-line descriptions, Types of Cooperatives per RA 9520)

3. **Legal Reference Card**
   Key RA 9520 provisions referenced in the training, with article/section numbers and plain-language summaries

4. **Checklist or Procedure Card**
   Step-by-step procedures taught in the training (e.g., Steps to register a cooperative, How to formulate MVO)

## Content Generation Guidelines

1. **Language**: Simple, direct, Grade 6-8 reading level. Avoid jargon unless it is defined in the Key Terms box for that section.
2. **Structure over prose**: Use bullet points, numbered lists, tables, and structured worksheets over long paragraphs.
3. **Adequate white space**: Participants need room to write. Do not pack pages too densely.
4. **Consistency**: Every topic section follows the same A-B-C-D pattern (Key Concepts, Key Terms, Activity Worksheet, Notes Space). This predictability helps participants navigate.
5. **Activity alignment**: Worksheets must match the activities described in the training design session plan. If the session plan says "MVO Formulation Workshop (60 min)", the handout must have a corresponding MVO worksheet.
6. **Self-contained**: The handout should be useful even after the training ends. Include enough context in Key Concepts and Reference Sheets that a participant can review the material independently.
7. **BARMM context**: Use examples relevant to BARMM cooperatives. Reference local cooperative types (fishing, farming, multi-purpose). Acknowledge multilingual context.
8. **Page breaks**: Start each new topic section on a fresh page. Keep worksheets on a single page where possible to avoid confusion.
