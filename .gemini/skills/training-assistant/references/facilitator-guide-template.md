# Facilitator Guide Template Structure

This template defines the standard structure for facilitator guides generated from CSEA-BARMM training designs. The facilitator guide is the most detailed deliverable in the training package -- it is the facilitator's complete playbook for delivering the training.

---

## Document Styling

- **Font**: Inter throughout
- **Body**: 10pt, line spacing 1.0, single extra line between paragraphs
- **Headings**: H1 = 18pt bold `#1B4F72`, H2 = 14pt bold `#1B4F72`, H3 = 12pt bold `#2E86C1`
- **Accent color**: `#1B4F72` (dark blue) for borders, headers, callout bars
- **Table headers**: `#1B4F72` background, white text
- **Alternating rows**: `#F8F9FA`
- **Icon markers**: Use Unicode symbols, never emojis
  - Time cue: [clock symbol] or the word TIME:
  - Warning/prep needed: [triangle warning] or the word PREP:
  - Tip: [lightbulb] or the word TIP:
  - Key point: [arrow right] or the word KEY:

### SECTION 5: ASSESSMENT ADMINISTRATION GUIDE

**5.1 Pre-Test Administration**

- Distribute pre-test forms at the designated time (per session plan)
- For participants with limited reading proficiency: read each item aloud in Filipino, provide local language explanation as needed
- Allow {X} minutes for completion
- Collect all forms before proceeding
- Do NOT review answers at this point

**5.2 Formative Assessment Notes**

For each formative assessment mentioned in the training design:
- What to observe during the activity
- Criteria for satisfactory performance
- How to provide in-the-moment feedback

**5.3 Post-Test Administration**

- Administer at the designated time
- Same accommodations as pre-test
- After collection, briefly review key answers if time permits
- Compare pre/post results to gauge learning gains

**5.4 Answer Key**

Include the complete answer key for all assessment items (from Training Design Section 15).

## Content Generation Guidelines

1. **Voice**: Second person, direct ("You will distribute...", "Ask the group to...")
2. **Specificity**: Avoid vague instructions like "facilitate a discussion." Instead: "Ask participants: 'What do you think is the main difference between a cooperative and a corporation?' Allow 2-3 responses before synthesizing."
3. **Time awareness**: Always include time cues. Facilitators need to know how long each sub-step should take.
4. **Preparation flags**: Any activity requiring advance preparation should be flagged in both the Preparation Checklist (Section 3) and in the session block where it appears.
5. **Transitions**: Every segment should end with a clear transition cue to the next segment.
6. **Adaptability**: Note where facilitators can adjust based on group size, energy level, or time constraints.
7. **BARMM context**: Reference local cooperative examples, use accessible language, and acknowledge the multilingual facilitation environment.
