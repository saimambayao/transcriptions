# General CSW Template (GN)

Universal template for Complete Staff Work that doesn't fit a specialized type (Budget Briefer, Legislative Briefer, Policy Recommendation). Follows ADDRESS IT directly as the section structure.

---

## Template Structure

### GN01 — EXECUTIVE SUMMARY

**ADDRESS IT Step**: 5 (Submit — write LAST)

Write after all other sections are complete. One page maximum. Captures: the problem, the recommended solution, expected benefits, and the action requested from the decision-maker.

---

### GN02 — BACKGROUND AND CONTEXT

**ADDRESS IT Step**: 1 (Analyze the Context)

- The assignment: what was asked, by whom, and why
- The office's function and the staff member's role in addressing it
- Stakeholders affected (primary and secondary)
- Internal and external factors (organizational, political, economic, social, technological)
- Relevant laws, policies, and regulations (BOL, BAAs, national laws, internal policies)
- Preliminary data gathered

---

### GN03 — PROBLEM OR NEED STATEMENT

**ADDRESS IT Step**: 2 (Define and Deliberate)

- Clear, concise articulation of the problem or need
- Root causes identified (distinguish from symptoms)
- Consequences of inaction
- Scope boundaries (what is and is not included)
- SMART objectives for the CSW effort
- Validated by relevant stakeholders

---

### GN04 — RESEARCH FINDINGS

**ADDRESS IT Step**: 3 (Research Needed Information)

- Research methodology: sources consulted, data gathered
- Key findings organized thematically
- Best practices and case studies identified
- Knowledge gaps acknowledged
- Visual aids (tables, charts) where appropriate
- All sources cited

---

### GN05 — OPTIONS ANALYSIS

**ADDRESS IT Step**: 4 (Explore Options)

- Options generated (minimum 2-3)
- Evaluation criteria: feasibility, effectiveness, efficiency, sustainability, alignment with organizational goals
- Impact analysis for each option
- Cost-benefit analysis where applicable
- Prioritized ranking with rationale

**Options Matrix Format**:

| Criterion | Option A | Option B | Option C |
|---|---|---|---|
| Feasibility | | | |
| Effectiveness | | | |
| Efficiency | | | |
| Sustainability | | | |
| Alignment | | | |
| **Overall Rank** | | | |

---

### GN06 — RECOMMENDED SOLUTION

**ADDRESS IT Step**: 4 (Explore — 4.8 Prepare for Recommendation) + Step 5 (Submit)

- The recommended option with clear rationale
- How it addresses the problem statement
- Alignment with organizational goals and BARMM development framework
- Supporting evidence and arguments
- Expected benefits and outcomes

---

### GN07 — IMPLEMENTATION PLAN

**ADDRESS IT Step**: 5 (Submit — 5.7) + Step 6 (Sustain)

- Specific actions, timelines, and milestones
- Resource requirements (personnel, budget, infrastructure)
- Roles and responsibilities
- Risk assessment and mitigation strategies
- Monitoring and evaluation mechanisms

---

### GN08 — CONCLUSION AND CALL TO ACTION

**ADDRESS IT Step**: 5 (Submit — 5.8)

- Summary of key points
- Benefits of implementing the recommendation
- Consequences of inaction
- Clear call to action: what the decision-maker should approve or decide

---

### GN09 — APPENDICES (as needed)

Supporting documents, detailed data tables, stakeholder consultation records, legal provisions, etc.

---

## Output Checklist

- [ ] All GN sections filled — no placeholders
- [ ] Problem statement is specific, measurable, and validated
- [ ] Research sources are credible and cited
- [ ] Options matrix includes at least 2-3 genuine alternatives (not strawmen)
- [ ] Recommendation is clearly justified with evidence
- [ ] Implementation plan includes timeline, resources, and risks
- [ ] Executive summary written last
- [ ] Transmittal memorandum prepared
- [ ] Names, titles, and legal citations verified
