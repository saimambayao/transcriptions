## Course Levels

| Level | Audience | Duration | Prerequisites |
|-------|----------|----------|---------------|
| **Foundation** | Business students, non-finance managers | 8-16 hours | Basic math, Excel basics |
| **Intermediate** | Finance professionals, analysts | 24-40 hours | Foundation course or equivalent |
| **Advanced** | Senior analysts, FP&A professionals | 40+ hours | Intermediate course |
| **Executive** | C-suite, board members | 8-16 hours | Business experience |

## Foundation Course

### Module 1: Introduction to Financial Statements (2 hours)

**Learning Outcomes:**
- Explain the purpose of financial statements
- Identify the three core financial statements
- Understand how statements connect

**Topics:**
1. Why Financial Statements Matter
2. The Three Core Statements Overview
3. Annual Reports and SEC Filings
4. Stakeholders and Their Information Needs

### Module 2: The Income Statement (2 hours)

**Learning Outcomes:**
- Read and interpret an income statement
- Calculate gross profit, operating income, net income
- Understand revenue recognition basics

**Topics:**
1. Revenue Recognition Concepts
2. Cost of Goods Sold (COGS)
3. Operating Expenses
4. Interest and Taxes
5. Reading Real-World Income Statements

### Module 3: The Balance Sheet (2 hours)

**Learning Outcomes:**
- Explain the accounting equation
- Identify assets, liabilities, and equity
- Understand working capital components

**Topics:**
1. The Accounting Equation (A = L + E)
2. Current Assets and Current Liabilities
3. Long-term Assets and Liabilities
4. Shareholders' Equity
5. Reading Real-World Balance Sheets

### Module 4: The Cash Flow Statement (2 hours)

**Learning Outcomes:**
- Distinguish between cash and accrual accounting
- Identify operating, investing, financing activities
- Reconcile net income to operating cash flow

**Topics:**
1. Cash vs. Accrual Accounting
2. Operating Activities
3. Investing Activities
4. Financing Activities
5. Free Cash Flow Introduction

### Module 5: Basic Ratio Analysis (4 hours)

**Learning Outcomes:**
- Calculate basic profitability ratios
- Calculate basic liquidity ratios
- Interpret ratios in context

**Topics:**
1. Why Ratios Matter
2. Profitability Ratios (Gross Margin, Net Margin, ROA, ROE)
3. Liquidity Ratios (Current, Quick)
4. Leverage Ratios (Debt-to-Equity)
5. Putting It Together: A Simple Company Analysis

## Intermediate Course

### Module 1: Financial Statement Analysis Deep Dive (4 hours)

**Learning Outcomes:**
- Apply common-size analysis
- Perform trend analysis
- Identify accounting quality issues

**Topics:**
1. Vertical Common-Size Analysis
2. Horizontal/Trend Analysis
3. Accounting Policies and Quality
4. Adjustments for Comparability
5. Case Study: Analyzing Company Financials

### Module 2: Liquidity and Activity Ratio Analysis (4 hours)

**Learning Outcomes:**
- Calculate and interpret all liquidity ratios
- Calculate and interpret activity ratios
- Analyze the cash conversion cycle

**Topics:**
1. Current, Quick, Cash Ratios
2. Defensive Interval Ratio
3. Inventory Turnover and DIO
4. Receivables Turnover and DSO
5. Payables Turnover and DPO
6. Cash Conversion Cycle Analysis

### Module 3: Profitability and Solvency Analysis (4 hours)

**Learning Outcomes:**
- Calculate and interpret profitability ratios
- Calculate and interpret solvency ratios
- Assess financial sustainability

**Topics:**
1. Gross, Operating, Net Margins
2. EBITDA and EBITDA Margin
3. ROA, ROE, ROIC
4. Debt-to-Equity, Debt-to-Capital
5. Interest Coverage and Fixed Charge Coverage
6. Comprehensive Case Study

### Module 4: DuPont Analysis and ROE Decomposition (3 hours)

**Learning Outcomes:**
- Perform 3-step DuPont analysis
- Perform 5-step DuPont analysis
- Identify drivers of ROE

**Topics:**
1. ROE as Key Performance Indicator
2. 3-Step DuPont Framework
3. 5-Step DuPont Framework
4. Comparing Companies Using DuPont
5. Case Study: DuPont Analysis

### Module 5: Introduction to Financial Modeling (4 hours)

**Learning Outcomes:**
- Understand financial model structure
- Build a simple projection model
- Apply best practices

**Topics:**
1. What is a Financial Model?
2. Model Structure: Inputs, Calculations, Outputs
3. Revenue Projection Methods
4. Expense Modeling
5. Building a Simple P&L Projection

### Module 6: Excel for Financial Analysis (4 hours)

**Learning Outcomes:**
- Use key Excel functions for finance
- Build organized, auditable spreadsheets
- Create dynamic financial analysis tools

**Topics:**
1. Essential Functions (SUMIF, VLOOKUP/XLOOKUP, INDEX/MATCH)
2. Financial Functions (NPV, IRR, PMT)
3. Data Tables and Sensitivity Analysis
4. Charts and Visualization
5. Model Organization and Best Practices

### Module 7: Working Capital Analysis (3 hours)

**Learning Outcomes:**
- Analyze working capital efficiency
- Calculate net working capital
- Optimize cash conversion

**Topics:**
1. Components of Working Capital
2. Net Working Capital Calculation
3. Working Capital Ratios Review
4. Cash Conversion Cycle Optimization
5. Industry Considerations

### Module 8: Trend Analysis and Forecasting (4 hours)

**Learning Outcomes:**
- Perform multi-year trend analysis
- Develop revenue and expense forecasts
- Build scenario analysis

**Topics:**
1. Historical Trend Analysis
2. Revenue Forecasting Methods
3. Cost Forecasting Methods
4. Scenario Analysis Basics
5. Building a Forecast Model

## Advanced Course

### Module 1: Advanced Financial Statement Analysis (6 hours)

**Topics:**
1. Quality of Earnings Analysis
2. Accounting Red Flags
3. Off-Balance Sheet Items
4. Non-GAAP Metrics
5. Forensic Accounting Basics
6. Case Study: Detecting Accounting Issues

### Module 2: Financial Modeling Best Practices (6 hours)

**Topics:**
1. Model Architecture and Design
2. Building Dynamic Models
3. Error Checking and Auditing
4. Scenario and Sensitivity Modeling
5. Model Documentation
6. Best Practice Case Study

### Module 3: Building Integrated Financial Models (8 hours)

**Topics:**
1. Three-Statement Model Overview
2. Building the Income Statement
3. Building the Balance Sheet
4. Building the Cash Flow Statement
5. Linking the Statements
6. Debt and Interest Schedules
7. Working Capital Schedules
8. Complete 3-Statement Model

### Module 4: Sensitivity and Scenario Analysis (4 hours)

**Topics:**
1. One-way Sensitivity Tables
2. Two-way Sensitivity Tables
3. Scenario Manager
4. Monte Carlo Simulation Basics
5. Presenting Sensitivity Results

### Module 5: Valuation Foundations (6 hours)

**Topics:**
1. Valuation Overview (→ /investor for deal context)
2. DCF Mechanics
3. Building a Simple DCF Model
4. Comparable Company Analysis Basics
5. Valuation Output Summary

### Module 6: Industry-Specific Analysis (4 hours)

**Topics:**
1. Retail/Consumer Analysis
2. Manufacturing/Industrial Analysis
3. Financial Services Analysis
4. Technology Analysis
5. Healthcare Analysis

### Module 7: Cash Flow Forecasting (4 hours)

**Topics:**
1. Direct vs. Indirect Cash Flow
2. Cash Flow Projection Methods
3. Working Capital Forecasting
4. CapEx and D&A Forecasting
5. Complete Cash Flow Model

### Module 8: Management Reporting and Dashboards (6 hours)

**Topics:**
1. Key Performance Indicators (KPIs)
2. Dashboard Design Principles
3. Building Excel Dashboards
4. Variance Analysis
5. Presentation and Storytelling
6. Case Study: Executive Dashboard

## Assessment Methods

| Assessment | Weight | Description |
|------------|--------|-------------|
| **Quizzes** | 20% | Module-end knowledge checks |
| **Practice Problems** | 20% | Ratio calculations, model exercises |
| **Case Studies** | 30% | Real company analysis |
| **Final Project** | 30% | Comprehensive financial analysis |

### Final Project Options

1. **Company Financial Health Assessment**: Full ratio analysis and recommendations
2. **Financial Model**: Build 3-statement model for company
3. **Industry Comparison**: Compare 3+ companies using learned techniques
