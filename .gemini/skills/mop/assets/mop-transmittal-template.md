## Usage Notes

- The transmittal memo wraps the complete MOP for formal submission to the approving authority
- The Background section should clearly state the trigger and justification for MOP development
- The Recommended Action must include both approval AND implementation directives (office order + orientation)
- For MOPs developed as part of a CSW assignment, this transmittal is used INSTEAD of the generic CSW transmittal -- the MOP is the CSW deliverable itself
- Include an Implementation Plan as an attachment covering: rollout timeline, orientation schedule, feedback collection period, and first review date
- The Endorsing Authority block is for the supervisor's sign-off before it reaches the final Approving Authority
