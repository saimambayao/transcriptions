# Y Combinator Framework for Enterprise Development

YC's methodology provides the transition framework from startup to scale.

## YC's Startup-to-Enterprise Transition

| YC Startup Phase | Enterprise Phase Transition |
|------------------|----------------------------|
| **PMF achieved** | Build systems for growth |
| **5-7% weekly** | Sustainable compounding |
| **Manual processes** | Automated operations |
| **Founder-driven** | Team-driven execution |
| **Do things that don't scale** | Build things that scale |

## YC Growth Metrics at Scale

| Stage | Startup (YC Phase) | Enterprise Phase |
|-------|-------------------|------------------|
| **Growth Target** | 5-7% weekly | 15-30% annually |
| **Focus** | Product-market fit | Market expansion |
| **Team** | Founders + few hires | Professional management |
| **Metrics** | User growth, retention | Revenue, margins, efficiency |

## When to Transition from /startup to /enterprise-dev

**YC Transition Criteria:**
- Sean Ellis 40%+ PMF achieved
- Repeatable customer acquisition
- Positive unit economics (or clear path)
- Core team in place
- Consistent revenue growth
- Processes need systematization
