## Community Management Fundamentals

### What is Community Management?

Community management is the practice of building authentic relationships between an organization and its audience through social media interactions.

### Why It Matters

| Statistic | Implication |
|-----------|-------------|
| 64% want brands to connect | Audience expects engagement |
| 88% see community impact on business | Communities drive results |
| Social listening essential | Must monitor conversations |

### Types of Community Management

| Type | Focus | Activities |
|------|-------|------------|
| **Customer Service** | Support | Answering questions, resolving issues |
| **Product Feedback** | Input | Beta testing, gathering suggestions |
| **Customer Acquisition** | Growth | Engaging prospects, generating leads |
| **Customer Engagement** | Loyalty | Building connections, fostering discussions |

### Community Manager Responsibilities

```
Daily Tasks:
□ Monitor mentions and comments
□ Respond to messages and DMs
□ Engage with community content
□ Share relevant content
□ Track sentiment

Weekly Tasks:
□ Review engagement metrics
□ Identify trending discussions
□ Report on community health
□ Plan engagement activities
□ Update response templates

Monthly Tasks:
□ Analyze community growth
□ Review and update guidelines
□ Identify top contributors
□ Plan community initiatives
□ Benchmark against goals
```

## Engagement Strategies

### Response Framework

| Interaction Type | Response Time | Approach |
|------------------|---------------|----------|
| **Positive comment** | Same day | Thank, engage, share |
| **Question** | Within 2 hours | Answer or escalate |
| **Complaint** | Within 1 hour | Acknowledge, resolve |
| **Negative review** | Within 4 hours | Address professionally |
| **Crisis** | Immediately | Follow crisis protocol |

### Engagement Best Practices

| Practice | Description |
|----------|-------------|
| **Be human** | Use conversational tone, show personality |
| **Be helpful** | Answer questions completely |
| **Be responsive** | Reply quickly and consistently |
| **Be grateful** | Thank supporters genuinely |
| **Be proactive** | Don't just react, initiate conversations |

### Conversation Starters

| Type | Examples |
|------|----------|
| **Questions** | "What's your experience with...?" |
| **Polls** | "Which do you prefer: A or B?" |
| **This or That** | "Coffee or tea?" |
| **Fill in the blank** | "My favorite thing about [topic] is ___" |
| **Throwbacks** | "Remember when...?" |
| **Behind-the-scenes** | "Here's what we're working on..." |

### Engagement Tactics

| Tactic | How to Implement |
|--------|------------------|
| **Comment first** | Leave thoughtful comments on related content |
| **Tag strategically** | Mention partners, customers, influencers |
| **Use Stories features** | Polls, questions, quizzes |
| **Go Live** | Real-time Q&A, events, updates |
| **Create Groups** | Facebook Groups for deeper community |
| **Reward engagement** | Highlight top contributors |

## Moderation Guidelines

### Community Guidelines Framework

| Section | Content |
|---------|---------|
| **Purpose** | What the community is for |
| **Expectations** | Acceptable behavior |
| **Prohibited** | What's not allowed |
| **Consequences** | What happens if rules broken |
| **Reporting** | How to flag issues |

### Prohibited Content Categories

| Category | Examples |
|----------|----------|
| **Hate speech** | Discrimination, slurs, attacks |
| **Harassment** | Bullying, threats, personal attacks |
| **Spam** | Repetitive posts, irrelevant promotions |
| **Misinformation** | False claims, fake news |
| **Illegal content** | Piracy, fraud, illegal activities |
| **Inappropriate** | Violence, explicit content |

### Moderation Approach

| Situation | Action |
|-----------|--------|
| **Minor violation** | Private warning, remove content |
| **Repeat offense** | Final warning, temp restriction |
| **Serious violation** | Immediate removal, permanent ban |
| **Legal issues** | Report to authorities, document |

### Response Templates

**Positive Feedback:**
```
Thank you so much for sharing this! We're so glad
to hear about your experience. [Personalized response]
```

**Question:**
```
Great question! [Answer]. If you have any other
questions, feel free to message us directly.
```

**Complaint (Public):**
```
We're sorry to hear about this experience. We'd like
to make it right. Please send us a message with your
details so we can help resolve this.
```

**Inappropriate Comment:**
```
We've removed this comment as it violates our community
guidelines. Please review our guidelines [link] to
understand what's acceptable in our community.
```

## Crisis Management

### What Constitutes a Crisis?

| Level | Characteristics | Examples |
|-------|-----------------|----------|
| **Minor** | Few mentions, contained | Single complaint, minor error |
| **Moderate** | Growing attention | Trending criticism, viral complaint |
| **Major** | Widespread, media attention | Scandal, serious incident, outage |

### Crisis Response Protocol

```
IMMEDIATE (0-1 hour):
□ Identify the issue
□ Pause scheduled content
□ Alert leadership/team
□ Gather facts
□ Draft initial response

SHORT-TERM (1-24 hours):
□ Post official response
□ Monitor all channels
□ Respond to individual queries
□ Update as new info available
□ Document everything

LONG-TERM (24+ hours):
□ Continue monitoring
□ Address ongoing concerns
□ Conduct post-mortem
□ Update protocols
□ Rebuild trust
```

### Crisis Communication Principles

| Principle | Application |
|-----------|-------------|
| **Speed** | Respond quickly, even if just to acknowledge |
| **Transparency** | Be honest about what happened |
| **Responsibility** | Take ownership when at fault |
| **Empathy** | Show you understand the impact |
| **Action** | Explain what you're doing about it |
| **Consistency** | Align internal and external messaging |

### What NOT to Do in a Crisis

| Mistake | Why It's Harmful |
|---------|------------------|
| **Delete and ignore** | Makes it worse, breaks trust |
| **Get defensive** | Escalates situation |
| **Blame others** | Looks unprofessional |
| **Use PR jargon** | Seems insincere |
| **Respond emotionally** | Can escalate conflict |
| **Lie or mislead** | Will be discovered, damages trust |

## Building Brand Advocates

### Advocate Development Funnel

```
FOLLOWER
    ↓ Engage consistently
ENGAGER
    ↓ Provide value, respond
FAN
    ↓ Recognize, reward
ADVOCATE
    ↓ Empower, involve
AMBASSADOR
```

### Identifying Potential Advocates

| Indicator | What to Look For |
|-----------|------------------|
| **Engagement** | Consistently likes, comments, shares |
| **Positive sentiment** | Speaks well of brand |
| **Influence** | Has own following |
| **Content creation** | Creates UGC about brand |
| **Referrals** | Brings others to community |

### Nurturing Advocates

| Strategy | Implementation |
|----------|----------------|
| **Recognition** | Feature their content, thank publicly |
| **Exclusive access** | Early previews, insider info |
| **Involvement** | Ask for input, include in decisions |
| **Rewards** | Discounts, freebies, special offers |
| **Community** | Create VIP group or program |

### Advocate Program Ideas

| Program Type | Description |
|--------------|-------------|
| **Brand Ambassador** | Formal program with expectations and rewards |
| **VIP Group** | Exclusive Facebook Group or community |
| **Referral Program** | Incentives for bringing new customers |
| **Content Creator** | Support UGC creation with resources |
| **Advisory Board** | Input on products/services |
