# BBSA Compliance Reference

> Extracted from `/budget` SKILL.md for detailed reference.

## BBSA Key Principles

| Principle | Description | Application |
|-----------|-------------|-------------|
| **Fiscal Autonomy** | BARMM control over revenues and expenditures | Block grant management |
| **Transparency** | Open budget processes and reporting | Public disclosure |
| **Accountability** | Clear responsibility for fund use | Audit trails |
| **Efficiency** | Optimal resource utilization | Performance-based budgeting |

## BBSA Compliance Checklist

```text
BBSA Compliance Requirements:

├── Budget Preparation
│   ├── [ ] Budget Call issued per schedule
│   ├── [ ] Agency submissions complete
│   ├── [ ] Public consultation conducted
│   └── [ ] Cabinet review completed
│
├── Budget Authorization
│   ├── [ ] BTA/Parliament deliberation
│   ├── [ ] Appropriations Act enacted
│   └── [ ] Budget published/disclosed
│
├── Budget Execution
│   ├── [ ] Allotments released timely
│   ├── [ ] Obligations within allotments
│   ├── [ ] Disbursements properly documented
│   └── [ ] Quarterly reports submitted
│
└── Accountability
    ├── [ ] COA audit requirements met
    ├── [ ] Financial statements prepared
    └── [ ] Performance reports submitted
```

## Variance Analysis

| Variance Type | Calculation | Action |
|---------------|-------------|--------|
| **Favorable** | Budget > Actual | Document savings, realignment opportunity |
| **Unfavorable** | Actual > Budget | Root cause analysis, augmentation request |
| **Timing** | Monthly deviation | Cash flow adjustment |
