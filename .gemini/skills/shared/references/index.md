## How to Reference

From any skill, reference these shared files using relative paths:

```markdown
# From /startup/SKILL.md:
See [YC Methodology](../shared/references/yc-methodology.md)

# From /cofounder/references/:
See [YC Co-Founder Dynamics](../../shared/references/yc-cofounder-dynamics.md)
```
