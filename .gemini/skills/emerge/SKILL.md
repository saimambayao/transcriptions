# Emerge — Surface the Unarticulated

Find ideas your vault implies but never states. This is the highest-value thinking tool — it reveals what you can't see yourself.

## Process

1. **Scan the vault broadly** — read notes across all folders (projects, research, daily, inbox, references). Focus on recurring themes, repeated phrases, and ideas that appear in multiple unrelated notes.

2. **Identify latent patterns**:
   - Concepts that keep appearing across different domains but are never named
   - Contradictions between stated beliefs and actual behavior (from daily notes)
   - Ideas that evolved silently over time without the user noticing
   - Connections between projects that share unstated assumptions
   - Recurring frustrations or desires that point to an unbuilt solution

3. **Present findings** as a numbered list of emergent insights:

For each insight:
- **Pattern**: what you observed across the vault
- **Evidence**: which notes contain the signal (with [[links]])
- **The unnamed idea**: what the vault is implying but the user hasn't articulated
- **So what**: why this matters or what the user might do with it

4. Keep it to **3-5 insights maximum** — quality over quantity. Each should be genuinely surprising, not obvious.

5. This is a read-only operation. Do not modify any files.
