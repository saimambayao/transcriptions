# Citation Format Standards

## Inline Citations

Use bracketed numbers immediately after the claim they support:

| Pattern | Usage | Example |
|---------|-------|---------|
| `[1]` | Single source | "Cooperatives grew 23% [1]" |
| `[1, 3]` | Multiple sources | "BARMM population is 4.4M [1, 3]" |
| `[1, p. 5]` | With page reference | "Section 3 requires registration [1, p. 5]" |
| `[1, Art. IV]` | With section reference | "Fiscal autonomy is guaranteed [1, Art. IV]" |
| `[3 vs 5]` | Conflicting sources | "Growth was 7.2% [3] or 6.8% [5]" |

Place the citation **after** the confidence tag when both are present:
```
Government cooperatives increased 23% [S][1, 3, 7]
```

## Works Cited List Format

Each entry follows this structure:
```
[N] Author/Organization. "Title." Source/Publisher, Year. URL
```

### Standard Source

```
[1] Philippine Statistics Authority. "2020 Census of Population and Housing." PSA, 2021. https://psa.gov.ph/census/2020
```

### No Author (Use Organization)

```
[2] Cooperative Development Authority. "Annual Report 2024." CDA, 2024. https://cda.gov.ph/reports/2024
```

### Academic Paper

```
[3] Santos, Maria L. and Cruz, Juan D. "Cooperative Governance in BARMM." Journal of Philippine Development, Vol. 48, No. 2, 2024. https://example.edu.ph/jpd/48-2
```

### Government Document / Legislation

```
[4] Republic of the Philippines. "Republic Act No. 9520: Philippine Cooperative Code of 2008." Official Gazette, 2009. https://www.officialgazette.gov.ph/2009/02/17/republic-act-no-9520/
[5] Bangsamoro Transition Authority. "Bangsamoro Autonomy Act No. 13: Bangsamoro Administrative Code." BTA Parliament, 2023.
```

### Web Page (No Date)

```
[6] World Bank. "Philippines Overview." Accessed March 2026. https://worldbank.org/en/country/philippines/overview
```

## Date Formats

| Context | Format | Example |
|---------|--------|---------|
| Publication year | `YYYY` | 2024 |
| Specific date | `Month DD, YYYY` | January 15, 2024 |
| Access date (no pub date) | `Accessed Month YYYY` | Accessed March 2026 |
| Date range | `YYYY-YYYY` | 2023-2028 |

## Numbering Rules

- Citations are numbered **sequentially** in order of first appearance
- Each unique source gets **one** number — reuse the same number for repeat citations
- Never skip numbers or leave gaps
- The Works Cited list must be ordered by citation number, not alphabetically
- Every inline citation `[N]` must have a corresponding Works Cited entry, and vice versa

## Source Type Labels

Use these type labels in the Source Log table:

| Label | Applies To |
|-------|-----------|
| `Official` | Government documents, laws, regulations, official reports |
| `Academic` | Peer-reviewed papers, university publications, theses |
| `Industry` | Standards bodies, industry reports (McKinsey, Gartner) |
| `News` | Established news outlets (Reuters, Philippine Star, Rappler) |
| `Org` | International organizations (UN, World Bank, ADB) |
| `Technical` | Official documentation, RFCs, technical specifications |

## Common Errors to Avoid

- **Orphaned citations**: Cited inline but missing from Works Cited (or vice versa)
- **Vague attribution**: "According to studies..." without a citation number
- **Broken URLs**: Always verify URLs resolve before including them
- **Wrong numbering**: Citation [3] in text but Works Cited jumps from [2] to [4]
- **Missing access dates**: Web sources without publication dates need "Accessed [Month Year]"
