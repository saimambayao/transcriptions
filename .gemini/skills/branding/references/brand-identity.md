## Kapferer Brand Identity Prism

Jean-Noël Kapferer's hexagonal prism model defines brand identity through 6 interconnected facets.

### The 6 Facets

```
            EXTERNALIZATION                    INTERNALIZATION
            (How brand appears)                (How brand feels)

            ┌─────────────────────────────────────────────────┐
            │                    SENDER                        │
            │   ┌───────────────────┬───────────────────┐     │
            │   │     PHYSIQUE      │   PERSONALITY     │     │
            │   │   (What it looks  │   (Character,     │     │
            │   │    like)          │    voice)         │     │
            │   ├───────────────────┼───────────────────┤     │
            │   │   RELATIONSHIP    │     CULTURE       │     │
            │   │   (Connection     │   (Values,        │     │
            │   │    with user)     │    inspiration)   │     │
            │   ├───────────────────┼───────────────────┤     │
            │   │    REFLECTION     │    SELF-IMAGE     │     │
            │   │   (Target user    │   (User's inner   │     │
            │   │    stereotype)    │    mirror)        │     │
            │   └───────────────────┴───────────────────┘     │
            │                    RECEIVER                      │
            └─────────────────────────────────────────────────┘
```

### Facet Definitions

| Facet | Question | Description |
|-------|----------|-------------|
| **Physique** | What does it look like? | Tangible elements: logo, colors, packaging |
| **Personality** | What character does it have? | Brand voice, attitude, communication style |
| **Culture** | What values does it embody? | Origins, beliefs, core values |
| **Relationship** | What connection does it create? | Transaction type, emotional bond |
| **Reflection** | Who is the typical user? | Outward mirror of target customer |
| **Self-Image** | How does using it make people feel? | Internal mirror, self-perception |

### BARMM Organization Examples

| Facet | Cooperative | Social Enterprise | Government (CSEA) |
|-------|-------------|-------------------|-------------------|
| **Physique** | Green/blue colors, community imagery | Impact colors, artisan products | Blue/gold, official seal |
| **Personality** | Friendly, trustworthy, community-minded | Passionate, purposeful, authentic | Professional, reliable, helpful |
| **Culture** | Cooperation, solidarity, democracy | Impact-first, innovation, sustainability | Service, integrity, development |
| **Relationship** | Partnership, shared ownership | Co-creation, impact partnership | Public service, support |
| **Reflection** | Community-oriented member | Conscious consumer, change-maker | Cooperative/SE leader |
| **Self-Image** | "I belong, I contribute" | "I make a difference" | "I'm building Bangsamoro" |

## Visual Identity Elements

### Core Visual Elements

| Element | Purpose | Key Considerations |
|---------|---------|-------------------|
| **Logo** | Primary brand identifier | Scalability, memorability, versatility |
| **Color Palette** | Emotional associations | Primary, secondary, neutral colors |
| **Typography** | Voice and readability | Headlines, body, accent fonts |
| **Imagery Style** | Visual storytelling | Photography, illustration, icons |
| **Layout System** | Consistency and hierarchy | Grids, spacing, alignment |

### Logo Types

| Type | Description | Best For |
|------|-------------|----------|
| **Wordmark** | Brand name in styled typography | Distinctive names |
| **Lettermark** | Initials or acronym | Long names |
| **Pictorial** | Symbol representing brand | Universal recognition |
| **Abstract** | Geometric or abstract form | Unique positioning |
| **Combination** | Symbol + wordmark | Flexibility |
| **Emblem** | Text within symbol/badge | Traditional, official |

### Color Psychology

| Color | Associations | Common Use |
|-------|--------------|------------|
| **Blue** | Trust, stability, professionalism | Finance, government, technology |
| **Green** | Growth, nature, health | Environmental, agriculture, wellness |
| **Yellow** | Optimism, warmth, attention | Retail, food, energy |
| **Red** | Energy, urgency, passion | Food, entertainment, sales |
| **Orange** | Friendly, confident, creative | Youth, creative, community |
| **Black** | Sophistication, luxury, power | Luxury, fashion, professional |

**Note**: Avoid purple as per platform guidelines.

### Typography Categories

| Category | Characteristics | Use Case |
|----------|-----------------|----------|
| **Serif** | Traditional, authoritative | Body text, formal |
| **Sans-serif** | Modern, clean | Digital, headings |
| **Display** | Decorative, distinctive | Logos, headlines |
| **Script** | Elegant, personal | Accents, signatures |
| **Monospace** | Technical, precise | Code, data |

## Verbal Identity Elements

### Brand Voice Framework

| Dimension | Spectrum | Questions |
|-----------|----------|-----------|
| **Tone** | Formal ←→ Casual | How serious or relaxed? |
| **Authority** | Expert ←→ Approachable | How much expertise to display? |
| **Character** | Professional ←→ Playful | How much personality? |
| **Emotion** | Reserved ←→ Expressive | How much feeling to convey? |

### Voice Attributes Template

| We Are | We Are Not |
|--------|------------|
| Friendly | Unprofessional |
| Knowledgeable | Condescending |
| Supportive | Pushy |
| Clear | Technical |
| Inspiring | Preachy |

### Messaging Hierarchy

| Level | Purpose | Example |
|-------|---------|---------|
| **Tagline** | Memorable summary | "Empowering Bangsamoro Communities" |
| **Mission** | Purpose statement | "To foster sustainable cooperatives..." |
| **Value Props** | Key benefits | "Affordable • Authentic • Impact-driven" |
| **Proof Points** | Evidence of claims | "100+ certified coops, 50,000 members" |
| **Stories** | Emotional connection | Member success stories |

### Naming Conventions

| Type | Description | Example |
|------|-------------|---------|
| **Descriptive** | Says what it does | "Bangsamoro Cooperative Authority" |
| **Suggestive** | Implies benefit | "GrowMoro" |
| **Abstract** | Created word | "Moronia" |
| **Founder** | Named after person | "Mama Rosa's Weaves" |
| **Acronym** | Initials | "CSEA" |

## Brand Identity Guidelines

### Guidelines Document Structure

```
1. BRAND OVERVIEW
   - Brand story
   - Mission and vision
   - Brand values

2. LOGO USAGE
   - Primary logo
   - Logo variations
   - Clear space
   - Minimum size
   - Incorrect usage

3. COLOR PALETTE
   - Primary colors (with codes)
   - Secondary colors
   - Color combinations
   - Color accessibility

4. TYPOGRAPHY
   - Primary typeface
   - Secondary typeface
   - Type hierarchy
   - Usage guidelines

5. IMAGERY
   - Photography style
   - Illustration style
   - Icon system
   - Image don'ts

6. APPLICATIONS
   - Stationery
   - Digital
   - Signage
   - Merchandise

7. TONE OF VOICE
   - Brand personality
   - Voice attributes
   - Writing guidelines
   - Example copy
```

### Common Guidelines Mistakes

| Mistake | Why It's a Problem | Solution |
|---------|-------------------|----------|
| **Too rigid** | Limits creativity | Provide principles, not just rules |
| **Too vague** | Inconsistent application | Include specific examples |
| **Not accessible** | Can't be found | Digital, searchable, shareable |
| **Out of date** | Doesn't reflect brand | Regular reviews and updates |
| **Logo-only focus** | Ignores voice and experience | Cover all touchpoints |
