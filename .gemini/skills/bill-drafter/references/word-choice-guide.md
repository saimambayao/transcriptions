# Legislative Word Choice Guide

Quick reference for precise word selection in Bangsamoro Parliament bills.

## Words and Phrases to Avoid

### Archaic Legalisms (never use)

| Avoid | Replace with |
|-------|-------------|
| hereby | (delete — adds nothing) |
| henceforth | from this date / from the effectivity of this Act |
| herein, hereinabove, hereinafter | in this Act / above / below |
| heretofore | before this Act / previously |
| hereunder | under this section / under this Act |
| whatsoever, whosoever, wheresoever | (delete — "any" suffices) |
| aforementioned, aforesaid | (delete or name the specific provision) |
| forthwith | immediately |
| thereof, therewith, thereto | of it / with it / to it (or restate the noun) |
| inter alia | among others / including |

### Redundant Doublets (use one word, not two)

| Avoid | Replace with |
|-------|-------------|
| null and void | void |
| full and complete | full OR complete |
| true and correct | true OR correct |
| each and every | each |
| sole and exclusive | sole OR exclusive |
| cease and desist | stop OR cease |
| power and authority | authority |
| rules and regulations | rules (unless technically different) |
| due and payable | due |
| made and entered into | made OR entered into |
| authorize and empower | authorize |
| alter or modify | amend |
| force and effect | effect |

### Wordy Phrases (simplify)

| Avoid | Replace with |
|-------|-------------|
| in the event that | if |
| in close proximity | near |
| at the time of | when |
| due to the fact that | because |
| for the purpose of | to |
| in order to | to |
| with respect to | about / regarding |
| in accordance with | under / per |
| notwithstanding anything to the contrary | despite / although |
| be deemed to be | is |
| is entitled to | may |
| is hereby authorized to | may |
| at such time as | when |
| during such time as | while |
| until such time as | until |
| in the event of | if |
| a period of time not less than | for at least |
| not later than | by |
| prior to | before |
| subsequent to | after |
| in excess of | more than |
| a sufficient number of | enough |
| is applicable to | applies to |
| provided, further, that | (use a separate sentence with "However") |

### Unnecessary Qualifiers (delete)

These add no legal meaning and may imply their absence elsewhere means something different:

- strictly (prohibited)
- absolutely (must)
- completely (null)
- positively, definitely, certainly
- under any circumstances, under all circumstances
- to the fullest extent

## Preferred Formulations

### For Prohibitions (strongest to weakest)

1. **"[Subject] shall not [action]"** — standard for BTA bills
2. **"[Subject] must not [action]"** — equally strong
3. **"[Subject] is prohibited from [action]"** — explicit prohibition
4. **"No [subject] may [action]"** — acceptable for prohibitions
5. *Avoid:* "should not", "will not", "cannot", "is not allowed to"

### For Obligations

1. **"[Subject] shall [action]"** — standard for BTA bills (consistent with existing BAAs)
2. **"[Subject] must [action]"** — equally strong (growing international preference)
3. *Avoid:* "should", "ought to", "is expected to", "is requested to"

### For Rights and Permissions

1. **"[Subject] may [action]"** — standard discretionary power
2. **"[Subject] is entitled to [action]"** — for established rights
3. **"[Subject] is authorized to [action]"** — for delegated authority
4. *Avoid:* "might", "could", "would be allowed to", "shall be authorized to"

### For Definitions

- **"means"** — exhaustive definition (X means A, B, and C — nothing else)
- **"includes"** — partial, illustrative definition (X includes A, B, and C — and possibly others)
- **"does not include"** — explicit exclusion
- *Avoid:* "shall be construed to mean", "is defined as and shall be construed to mean", "shall mean"

### For Exceptions

- **"except"** — when the exception follows the rule in the same sentence
- **"unless"** — when the exception precedes the rule or is in a separate sentence
- **"Provided, That"** — BTA convention for inline exceptions (capitalize P and T, italicize)
- **"however"** — for a new sentence stating the exception
- *Avoid:* provisos when a separate subsection or sentence is clearer

### For Time References

- **"within [N] days"** — clear deadline from a triggering event
- **"not later than [date]"** → use **"by [date]"** instead
- **"upon"** — when action is triggered by an event ("upon receipt of the application")
- *Avoid:* "at the earliest possible time", "as soon as practicable" (too vague for normative provisions — specify days)

### For Amounts

- Write amounts in words AND figures: "Five Million Pesos (PHP 5,000,000.00)"
- This is BTA standard (see BAA-20, Section 9)
- Spell out numbers one through nine; use figures for 10 and above
- Exception: use figures for section numbers, percentages, dates

## BTA-Specific Conventions

### Dash After Section Title
BTA bills use an en-dash (–) after the section title period, not a colon or hyphen:
```
Section 1. Short Title. – This Act shall be known as...
```

Note: Some BTA bills use em-dash (—). Either is acceptable; be consistent within a single bill.

### Defined Terms
Bold defined terms in the Definition of Terms section:
```
(a) **"Covered Entities"** – refers to all ministries...
```

### Foreign Words
Italicize non-English terms (Arabic, Maranao, Maguindanaon):
```
*Hijri*, *fatwa*, *Darul-Ifta'*, *Shari'ah*
```

### Prophet Muhammad Reference
Include the honorific: "Prophet Muhammad (peace be upon him)" on first mention, or use the Arabic symbol (ﷺ) if the bill follows Islamic convention as in BAA-20.

### Enacting Clause
Always use the exact formulation:
```
Be it enacted by the Bangsamoro Transition Authority in Parliament assembled:
```

### Closing
```
APPROVED.

(Signature)
[SPEAKER NAME]
Speaker
```
