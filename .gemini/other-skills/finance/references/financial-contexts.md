## Cooperative Finance

### Unique Financial Characteristics

Cooperatives have distinct financial features based on cooperative principles:

| Characteristic | Description |
|----------------|-------------|
| **Member Ownership** | Members own and control the cooperative |
| **Patronage-Based Returns** | Profits distributed based on use, not investment |
| **Democratic Control** | One member, one vote (not based on capital) |
| **Limited Return on Capital** | Caps on returns to invested capital |

### Equity Structure

**Types of Cooperative Equity:**

| Type | Description |
|------|-------------|
| **Member Shares** | Initial investment to join cooperative |
| **Patronage Equity** | Retained earnings allocated to members |
| **Unallocated Equity** | Reserves not assigned to specific members |
| **Per-Unit Retains** | Deductions per unit of commodity handled |

### Patronage Refunds

**How Patronage Works:**
```
1. Cooperative earns net income
         ↓
2. Board allocates income based on member patronage
         ↓
3. Allocation distributed as:
   - Cash payment (typically 20-30%)
   - Retained equity (70-80%)
         ↓
4. Retained equity eventually redeemed
```

**Qualified vs. Nonqualified Patronage:**
| Type | Tax Treatment |
|------|---------------|
| **Qualified** | Member pays tax in year allocated |
| **Nonqualified** | Cooperative pays tax; member pays when redeemed |

### Equity Redemption Methods

| Method | Description |
|--------|-------------|
| **Revolving Fund** | Oldest equity redeemed first (FIFO) |
| **Age of Patron** | Redeem when member reaches certain age |
| **Percentage** | Redeem fixed percentage annually |
| **Special Allocation** | Board discretion based on finances |

### Financial Management Objectives

1. **Adequate Funding** - Sufficient capital for operations
2. **Member Returns** - Fair patronage refunds
3. **Equity Balance** - Between retention and redemption
4. **Financial Stability** - Long-term sustainability
5. **Member Accountability** - Transparent reporting

### Key Financial Statements for Cooperatives

| Statement | Unique Elements |
|-----------|-----------------|
| **Balance Sheet** | Member equity accounts, patronage payable |
| **Income Statement** | Patronage allocations, per-unit retains |
| **Equity Statement** | Changes in member equity accounts |
| **Cash Flow** | Patronage payments, equity redemptions |

### Challenges in Cooperative Finance

| Challenge | Consideration |
|-----------|---------------|
| **Capital Retention** | Balance between redemption and investment needs |
| **Member Expectations** | Members want quick equity redemption |
| **Horizon Problem** | Older members prefer immediate returns |
| **Free Rider Problem** | New members benefit from past investments |
| **Limited External Capital** | Cannot sell equity to non-members |

## Social Enterprise Finance

### Dual Bottom Line

Social enterprises manage both:
1. **Financial Sustainability** - Revenue > Expenses
2. **Social Impact** - Mission achievement

### Blended Value

> "Value that is a blend of economic, environmental and social factors, where maximizing value requires taking all three elements into account." — Jed Emerson

**Blended Value Framework:**
```
Total Value = Economic Value + Social Value + Environmental Value
```

### Revenue Models

| Model | Description |
|-------|-------------|
| **Fee-for-Service** | Charge for products/services provided |
| **Cross-Subsidy** | Profitable activities fund mission work |
| **Membership/Subscription** | Regular fees from members/supporters |
| **Hybrid Grants + Earned** | Blend of donations and revenue |
| **Social Franchising** | License model to expand impact |

### Financial Sustainability Strategies

| Strategy | Description |
|----------|-------------|
| **Revenue Diversification** | Multiple income streams |
| **Earned Income** | Products/services generating revenue |
| **Cost Efficiency** | Lean operations without compromising impact |
| **Reserve Building** | Cash reserves for stability |
| **Strategic Partnerships** | Resource sharing, joint ventures |

### Social Return on Investment (SROI)

**Definition:** Measures broader social, environmental, and economic value compared to investment.

**SROI Calculation:**
```
SROI Ratio = Present Value of Social Benefits / Value of Investment
```

**Example:** SROI of 3:1 means $3 of social value created for every $1 invested.

**SROI Principles (Social Value International):**
1. Involve stakeholders
2. Understand what changes
3. Value things that matter
4. Only include what is material
5. Do not over-claim
6. Be transparent
7. Verify results
8. Be responsive

### Impact Measurement

| Metric Type | Examples |
|-------------|----------|
| **Outputs** | Number served, products sold, hours delivered |
| **Outcomes** | Skills gained, income increased, health improved |
| **Impact** | Net change attributable to intervention |

### Mission Drift Prevention

**Financial Safeguards:**
- Mission-first budgeting
- Impact metrics in financial reports
- Board oversight of mission alignment
- Mission-locked governance documents
- Impact investment over profit maximization

### Funding Sources

| Source | Characteristics |
|--------|-----------------|
| **Grants** | Restricted use, reporting requirements |
| **Donations** | Flexible but unpredictable |
| **Earned Revenue** | Sustainable but requires capacity |
| **Impact Investment** | Patient capital expecting returns + impact |
| **Blended Finance** | Mix of philanthropic and commercial capital |

## Comparative Summary

### Financial Focus by Context

| Aspect | Cooperative | Social Enterprise |
|--------|-------------|-------------------|
| **Primary Goal** | Member benefit | Social mission |
| **Profit Distribution** | Based on patronage | Reinvested in mission |
| **Equity Source** | Members | Diverse (grants, investment, earned) |
| **Return Expectations** | Limited returns | Impact + modest financial return |
| **Accountability** | To members | To stakeholders + beneficiaries |

### Common Elements

Both cooperatives and social enterprises share:
1. **Mission Orientation** - Purpose beyond profit
2. **Stakeholder Focus** - Beyond just shareholders
3. **Community Benefit** - Positive social impact
4. **Sustainability** - Long-term viability matters
5. **Transparency** - Accountability to stakeholders

### Financial Reporting Considerations

| Requirement | Cooperative | Social Enterprise |
|-------------|-------------|-------------------|
| **Financial Statements** | Standard + patronage details | Standard + impact metrics |
| **Member/Stakeholder Reports** | Patronage statements | Impact reports |
| **Regulatory Reporting** | Cooperative registrar | Varies by structure |
| **Audit Requirements** | Varies by size/law | Varies by funding sources |
