## Training Levels

### Four-Tier Framework

| Level | Target Audience | Duration | Focus |
|-------|-----------------|----------|-------|
| **Foundation** | Board members, new managers, staff | 8-16 hours | Financial literacy, reading statements |
| **Intermediate** | Managers, treasurers, finance staff | 24-40 hours | Analysis, budgeting, controls |
| **Advanced** | Finance managers, executives | 40+ hours | Strategy, forecasting, capital management |
| **Executive** | C-suite, board chairs | 16-24 hours | Strategic finance, oversight |

### Level Objectives

**Foundation:**
- Read and understand financial statements
- Know basic accounting concepts
- Understand budgeting basics
- Recognize financial red flags

**Intermediate:**
- Conduct ratio analysis
- Prepare and manage budgets
- Implement internal controls
- Perform variance analysis

**Advanced:**
- Develop financial strategy
- Manage capital structure
- Lead financial planning
- Optimize cash flow

**Executive:**
- Provide financial oversight
- Align finance with strategy
- Evaluate financial performance
- Make capital decisions

## Foundation Finance Course

### Standard Duration: 8-16 hours (1-2 days)

### Module Outline

| Module | Topics | Duration |
|--------|--------|----------|
| **1. Financial Basics** | Accounting concepts, financial vs. management accounting | 1-2 hours |
| **2. Income Statement** | Revenue, expenses, profitability | 2-3 hours |
| **3. Balance Sheet** | Assets, liabilities, equity | 2-3 hours |
| **4. Cash Flow Statement** | Operating, investing, financing activities | 2-3 hours |
| **5. Budgeting Basics** | Budget types, preparation, monitoring | 1-2 hours |
| **6. Internal Controls** | Purpose, segregation of duties, fraud prevention | 1-2 hours |

### Learning Objectives

By end of course, participants can:
1. Define key accounting terms
2. Read and interpret three financial statements
3. Distinguish between cash and accrual accounting
4. Understand budget components
5. Explain importance of internal controls

### Key Concepts

**Accounting Equation:**
```
Assets = Liabilities + Equity
```

**Profit vs. Cash:**
| Profit (Accrual) | Cash |
|------------------|------|
| Revenue earned | Cash received |
| Expenses incurred | Cash paid |
| May differ from cash | Actual money movement |

## Intermediate Finance Course

### Standard Duration: 24-40 hours (3-5 days)

### Module Outline

| Module | Topics | Duration |
|--------|--------|----------|
| **1. Financial Analysis** | Ratio analysis, trend analysis, benchmarking | 4-6 hours |
| **2. Cash Flow Management** | Working capital, liquidity, cash forecasting | 4-6 hours |
| **3. Budgeting & Planning** | Budget preparation, types, variance analysis | 4-6 hours |
| **4. Cost Management** | Fixed/variable costs, break-even, cost control | 3-4 hours |
| **5. Internal Controls** | Control design, implementation, monitoring | 3-4 hours |
| **6. Financial Reporting** | Preparation, compliance, transparency | 3-4 hours |

### Learning Objectives

By end of course, participants can:
1. Calculate and interpret financial ratios
2. Manage working capital effectively
3. Prepare and monitor budgets
4. Perform break-even analysis
5. Design appropriate internal controls
6. Prepare financial reports

### Module Details

**Module 1: Financial Analysis**
| Ratio Category | Key Ratios |
|----------------|------------|
| Liquidity | Current ratio, quick ratio |
| Profitability | Gross margin, ROA, ROE |
| Efficiency | Asset turnover, inventory turnover |
| Solvency | Debt-to-equity, interest coverage |

**Module 3: Budgeting & Planning**
| Budget Type | Purpose |
|-------------|---------|
| Operating Budget | Day-to-day expenses |
| Capital Budget | Long-term investments |
| Cash Budget | Cash flow planning |
| Master Budget | Comprehensive plan |

## Advanced Finance Course

### Standard Duration: 40+ hours (5+ days)

### Module Outline

| Module | Topics | Duration |
|--------|--------|----------|
| **1. Strategic Finance** | Finance strategy, value creation | 6-8 hours |
| **2. Capital Management** | Capital structure, funding options | 6-8 hours |
| **3. Financial Planning** | Long-range planning, forecasting | 6-8 hours |
| **4. Performance Management** | KPIs, dashboards, reporting | 4-6 hours |
| **5. Risk Management** | Financial risk, mitigation strategies | 4-6 hours |
| **6. Context-Specific Finance** | Cooperative/SE finance specifics | 6-8 hours |

### Learning Objectives

By end of course, participants can:
1. Develop financial strategy aligned with mission
2. Make capital structure decisions
3. Create multi-year financial plans
4. Design performance management systems
5. Identify and mitigate financial risks
6. Apply context-specific financial practices

### Module Details

**Module 2: Capital Management**
| Funding Source | Considerations |
|----------------|----------------|
| Member equity | Patronage, redemption schedules |
| Retained earnings | Reinvestment vs. distribution |
| Debt financing | Leverage, interest costs |
| Grants | Restrictions, compliance |

**Module 6: Context-Specific**
| For Cooperatives | For Social Enterprises |
|------------------|------------------------|
| Patronage accounting | SROI measurement |
| Equity redemption | Blended value |
| Member capital | Impact-financial balance |
| Democratic governance | Mission drift prevention |

## Assessment Methods

### Knowledge Assessment

| Method | Purpose |
|--------|---------|
| Pre-test | Baseline knowledge |
| Post-test | Learning gain |
| Case analysis | Application skills |
| Financial scenarios | Decision-making |

### Skills Assessment

| Method | Purpose |
|--------|---------|
| Financial statement exercises | Reading and analysis |
| Ratio calculation | Technical competency |
| Budget preparation | Planning skills |
| Variance analysis | Analytical ability |

### Certification

| Certificate | Requirements |
|-------------|--------------|
| Foundation Certificate | Attendance + post-test (70%+) |
| Intermediate Certificate | Attendance + assessments + case study |
| Advanced Certificate | Program + financial project + presentation |
| Executive Certificate | Program + strategic recommendation |

## Integration with /academy

### When to Use with /academy

Invoke `/finance` with `/academy` when:
1. Creating financial literacy training
2. Developing treasurer/finance officer programs
3. Building board financial oversight training
4. Designing financial management certification

### Workflow

```
1. IDENTIFY training need and audience
         ↓
2. SELECT course level (Foundation/Intermediate/Advanced)
         ↓
3. SELECT context (Cooperative/Social Enterprise)
         ↓
4. USE /finance for content expertise
         ↓
5. USE /academy for curriculum design
         ↓
6. DEVELOP learning materials
         ↓
7. IMPLEMENT training
         ↓
8. EVALUATE and improve
```

### Information to Provide /academy

When invoking with `/academy`:
1. **Course Level**: Foundation, Intermediate, Advanced, Executive
2. **Target Audience**: Board, managers, staff, treasurers
3. **Duration**: Hours/days available
4. **Delivery Mode**: In-person, online, hybrid
5. **Specific Topics**: If not full curriculum
6. **Context**: Cooperative, Social Enterprise, or both
7. **Assessment Requirements**: Certification needs

### Context-Specific Curriculum Adaptations

**For Cooperatives:**
- Include patronage accounting
- Cover equity redemption
- Add member capital concepts
- Focus on democratic financial oversight

**For Social Enterprises:**
- Include SROI methodology
- Cover blended value reporting
- Add impact-financial balance
- Focus on sustainability metrics
