# COOP-301: Financial Management for Cooperatives

Advanced financial management training for cooperative treasurers and managers.

---

## Overview

| Attribute | Value |
|-----------|-------|
| **Code** | COOP-301 |
| **Type** | Online Course (Self-Paced) |
| **Duration** | 8 hours |
| **Target Audience** | Treasurers, Finance Committee, CEO/GM |
| **Prerequisites** | COOP-101: Fundamentals of Cooperatives |
| **Compliance** | CDA MC 2015-09 (Required for coops with ₱5M+ deposit liabilities; adopted by CSEA for BARMM) |
| **Certificate** | Yes (80% completion + 70% assessment) |
| **Languages** | English + Filipino (bilingual scripts) |

## Learning Objectives

By the end of this course, learners will be able to:

1. **Read** and **interpret** cooperative financial statements
2. **Apply** PEARLS financial ratio analysis
3. **Prepare** and **monitor** annual budgets
4. **Manage** cash flow and liquidity
5. **Understand** investment options for cooperatives

## Script Requirements

### Video Scripts

Each video lesson requires:
1. **English Script** - Primary language for formal terms
2. **Filipino Script** - Localized version for accessibility
3. **Timing Markers** - Scene/slide change indicators
4. **Visual Cues** - On-screen text and graphics notes

### Script Pacing (~90 WPM for All Languages)

All scripts (Filipino or English) use **~90 wpm** to accommodate BARMM's ESL beginner learners. Additional pacing considerations:
- Longer words with affixes (mag-, nag-, -in, -an) in Filipino
- Technical term explanation pairs (English term + Filipino meaning)
- Natural code-switching (Taglish) pauses
- Clearer enunciation for educational content

**Technical Term Pattern**: `"[English term], o [Filipino explanation]"`
- Example: "Ang **balance sheet**, o talaang pinansyal, ay..."
- Example: "I-compute natin ang **PEARLS ratio**, ang sukat ng kalusugan pinansyal..."

**Timing Buffers**:
- Add +15 seconds per technical definition
- Add +20 seconds for calculation walkthroughs
- Total buffer: 15-20% of estimated duration

### Key PEARLS Facts

**History**: WOCCU developed PEARLS in the 1980s, adapting the CAMEL banking system specifically for credit unions and financial cooperatives.

**The PEARLS Acronym**:
- **P**rotection - Loan loss provisions, bad debt coverage
- **E**ffective Financial Structure - Asset allocation, equity structure
- **A**sset Quality - Delinquency, non-performing assets
- **R**ates of Return and Costs - ROA, ROE, operating efficiency
- **L**iquidity - Reserve ratios, cash management
- **S**igns of Growth - Asset, membership, loan growth

**Key Standards**:
- Administrative costs: 3-10% of total assets
- Liquidity reserves: 15-20% of deposits
- 45 financial ratios for comprehensive analysis
