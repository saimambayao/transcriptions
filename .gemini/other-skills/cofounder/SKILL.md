### Cofounder Compatibility Factors

| Factor | Why It Matters |
|--------|---------------|
| **Complementary skills** | Cover all needed competencies |
| **Shared vision** | Aligned on destination |
| **Compatible values** | Similar ethical framework |
| **Trust** | Foundation of partnership |

| Archetype | Strengths | Common Roles |
|-----------|-----------|--------------|
| **Hustler** | Sales, networking, deals | CEO, BD |
| **Hacker** | Technical, building | CTO, CPO |
| **Hipster** | Design, UX, branding | CDO, CMO |

### Equity Split Principles

| Model | When to Use |
|-------|-------------|
| **Equal split** | Similar contributions, high trust |
| **Dynamic equity** | Uncertain contributions over time |
| **Fixed unequal** | Clear difference in contribution |
| **Vesting** | Always - protect against early departure |

## Core Frameworks

### The Founder's Dilemmas (Noam Wasserman)

**Key Insights:**

| Dilemma | Trade-off |
|---------|-----------|
| **Rich vs. King** | Wealth vs. control |
| **Solo vs. Team** | Full control vs. capability |
| **Relationships vs. Skills** | Trust vs. complementarity |
| **Quick vs. Slow** | Speed vs. deliberation |

**Cofounder Equity Decision Tree:**

```text
EQUITY SPLIT DECISION:
│
├── Are contributions roughly equal?
│   ├── YES → Consider equal split
│   └── NO → Identify contribution differences
│
├── Are contributions known upfront?
│   ├── YES → Fixed split with vesting
│   └── NO → Consider dynamic equity (Slicing Pie)
│
├── Is there a clear "first" founder?
│   ├── YES → Founder premium (5-15%)
│   └── NO → Start from equal
│
└── Have you discussed failure scenarios?
    ├── YES → Document in agreement
    └── NO → Discuss before finalizing
```

### Y Combinator Cofounder Framework

YC's cofounder methodology is deeply integrated into this skill. See shared references for comprehensive YC content.

#### YC's Core Position: Equal Equity Splits

> "YC strongly recommends equal equity splits." — YC Partner Advice

| Why Equal | Explanation |
|-----------|-------------|
| **Trust Signal** | Shows co-founders value each other equally |
| **Simplifies Decisions** | No "I own more" power dynamics |
| **Long-term Motivation** | Everyone stays fully invested |
| **Reduces Conflict** | Nothing to argue about |

#### YC Standard: 4-Year Vesting, 1-Year Cliff

```
Year 0-1:    Nothing vests (cliff period)
Year 1:      25% vests immediately
Year 1-4:    Remaining 75% vests monthly
```

#### YC Founder Qualities (What YC Evaluates)

| Quality | Description |
|---------|-------------|
| **Determination** | Won't quit when things get hard |
| **Flexibility** | Willing to change based on evidence |
| **Domain Expertise** | Deep knowledge of the problem space |
| **Technical Ability** | Can build the product themselves |
| **Clarity** | Can explain idea in one sentence |

### Additional YC Cofounder Principles

1. **Work with people you know** - Not strangers from matching sites
2. **Test before committing** - Work on a project first
3. **Complementary skills matter** - But trust matters more
4. **Equity should reflect commitment** - Not just ideas
5. **Vesting protects everyone** - Always use vesting

|-----------|----------|--------------|
