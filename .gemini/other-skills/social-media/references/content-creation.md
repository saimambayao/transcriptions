## Content Types

### Content Categories

| Category | Purpose | Examples |
|----------|---------|----------|
| **Educational** | Teach and inform | Tutorials, tips, how-tos, explainers |
| **Entertaining** | Engage and delight | Memes, trends, humor, challenges |
| **Inspirational** | Motivate and uplift | Success stories, quotes, testimonials |
| **Promotional** | Drive action | Product features, offers, CTAs |
| **Community** | Build connection | UGC, polls, Q&A, events |
| **Behind-the-scenes** | Humanize brand | Team, process, day-in-life |

### Content Format Performance

| Format | Engagement Level | Best For |
|--------|------------------|----------|
| **Short-form video** | Highest | Awareness, entertainment |
| **Carousels** | Very high | Education, storytelling |
| **Single images** | High | Quick updates, quotes |
| **Stories** | High | Daily engagement, urgency |
| **Long-form video** | Medium-high | Tutorials, deep dives |
| **Text posts** | Medium | Announcements, discussions |
| **Links** | Lower | Traffic driving |

### Video Content Statistics

- 89% of marketers increasing video output
- 50% of time on Instagram/Facebook spent on Reels
- 69% of video watched without sound (add captions)
- Short-form video has highest ROI

| Format | Specs | Best Practices |
|--------|-------|----------------|
| **Feed Video** | Up to 240 mins, 4GB | 1-3 mins optimal, captions essential |
| **Reels** | Up to 90 secs | Vertical 9:16, trending audio |
| **Image Post** | 1200x630px | High quality, minimal text |
| **Carousel** | Up to 10 images | Swipe-worthy, educational |
| **Stories** | 9:16 vertical | 24-hour urgency, interactive |
| **Live** | Up to 8 hours | Announce ahead, engage viewers |

### Instagram

| Format | Specs | Best Practices |
|--------|-------|----------------|
| **Reels** | 15-90 secs, 9:16 | Trending audio, hooks in first 3 secs |
| **Feed Post** | 1080x1080 (square) | High-quality visuals |
| **Carousel** | Up to 10 slides | Educational, storytelling |
| **Stories** | 9:16 vertical | Polls, questions, stickers |

### TikTok

| Format | Specs | Best Practices |
|--------|-------|----------------|
| **Videos** | 15 secs - 10 mins | 15-60 secs optimal |
| **Stories** | 15 secs | Quick updates |

## Content Creation Process

### Creation Workflow

```
1. PLAN
   ├── Review content calendar
   ├── Identify content pillar
   ├── Define goal and CTA
   └── Research trends/topics

2. CREATE
   ├── Draft copy
   ├── Create/source visuals
   ├── Add captions/subtitles
   └── Optimize for platform

3. REVIEW
   ├── Check brand voice
   ├── Verify accuracy
   ├── Test accessibility
   └── Get approval if needed

4. SCHEDULE
   ├── Choose optimal time
   ├── Add hashtags/tags
   ├── Set up tracking
   └── Schedule or publish

5. ENGAGE
   ├── Monitor comments
   ├── Respond promptly
   ├── Track performance
   └── Gather insights
```

### Copywriting Best Practices

| Element | Guidelines |
|---------|------------|
| **Hook** | Capture attention in first line |
| **Value** | Deliver on promise quickly |
| **Voice** | Consistent brand personality |
| **CTA** | Clear next action |
| **Length** | Platform-appropriate (shorter = better) |
| **Hashtags** | 3-5 relevant, mix of sizes |

### Visual Content Guidelines

| Principle | Application |
|-----------|-------------|
| **Consistency** | Use brand colors, fonts, style |
| **Quality** | High resolution, well-lit |
| **Accessibility** | Alt text, color contrast |
| **Mobile-first** | Readable on small screens |
| **Authenticity** | Real people, genuine moments |

## Accessibility Guidelines

### Making Content Accessible

| Element | Best Practice |
|---------|---------------|
| **Captions** | Add to all videos |
| **Alt text** | Describe images |
| **Color contrast** | 4.5:1 minimum ratio |
| **Hashtags** | CamelCase (#SocialMedia) |
| **Plain language** | Avoid jargon |
| **Font size** | Large enough to read on mobile |

### Why Accessibility Matters

- Reaches wider audience
- Required for government content
- Improves engagement for all users
- Demonstrates inclusion values
