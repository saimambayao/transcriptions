## Key Performance Indicators

### Top 5 Social Media KPIs

| KPI | Definition | Formula |
|-----|------------|---------|
| **Engagement Rate** | Interaction level relative to audience | (Likes + Comments + Shares) / Followers × 100 |
| **Reach** | Unique users who saw content | Platform-reported metric |
| **Click-Through Rate** | Clicks on links/CTAs | Clicks / Impressions × 100 |
| **Follower Growth Rate** | Audience growth speed | (New Followers / Total Followers) × 100 |
| **Conversion Rate** | Actions taken from social | Conversions / Clicks × 100 |

### KPI Categories

| Category | Metrics | Purpose |
|----------|---------|---------|
| **Awareness** | Reach, impressions, share of voice | Brand visibility |
| **Engagement** | Likes, comments, shares, saves | Audience connection |
| **Traffic** | Clicks, website visits, bounce rate | Driving action |
| **Conversion** | Leads, sales, sign-ups | Business results |
| **Loyalty** | Repeat engagement, NPS, advocates | Long-term value |

### Engagement Rate Benchmarks

| Platform | Average | Good | Excellent |
|----------|---------|------|-----------|
| **Facebook** | 0.5-1% | 1-2% | 3%+ |
| **Instagram** | 1-3% | 3-5% | 6%+ |
| **TikTok** | 3-5% | 5-10% | 15%+ |
| **LinkedIn** | 0.5-1% | 1-2% | 3%+ |

**Note**: Micro-influencers typically see higher engagement (3-5% on Instagram vs 1.2% for macro-influencers).

## Metrics by Goal

### Brand Awareness Metrics

| Metric | What It Measures | How to Track |
|--------|------------------|--------------|
| **Impressions** | Total times content displayed | Platform analytics |
| **Reach** | Unique users who saw content | Platform analytics |
| **Share of Voice** | Brand mentions vs competitors | Social listening tools |
| **Brand Mentions** | Times brand is mentioned | Social listening |
| **Follower Count** | Total audience size | Platform analytics |

### Engagement Metrics

| Metric | What It Measures | Calculation |
|--------|------------------|-------------|
| **Likes/Reactions** | Positive response | Count |
| **Comments** | Conversation depth | Count |
| **Shares/Retweets** | Content spread | Count |
| **Saves** | Content value | Count |
| **Video Views** | Video engagement | Count (varies by platform) |
| **Video Completion** | Content quality | % watched to end |

### Traffic Metrics

| Metric | What It Measures | Tool |
|--------|------------------|------|
| **Link Clicks** | Interest in more info | Platform analytics |
| **Website Sessions** | Social-driven visits | Google Analytics |
| **Pages per Session** | Engagement depth | Google Analytics |
| **Bounce Rate** | Content relevance | Google Analytics |
| **Time on Site** | Content quality | Google Analytics |

### Conversion Metrics

| Metric | What It Measures | Calculation |
|--------|------------------|-------------|
| **Conversion Rate** | Action completion | Conversions / Clicks × 100 |
| **Cost per Click (CPC)** | Ad efficiency | Ad Spend / Clicks |
| **Cost per Acquisition (CPA)** | Customer cost | Ad Spend / Conversions |
| **Lead Generation** | Prospect capture | Form submissions |
| **Sales from Social** | Revenue impact | Tracked purchases |

### Customer Feedback Metrics

| Metric | What It Measures | Scale |
|--------|------------------|-------|
| **Net Promoter Score (NPS)** | Loyalty likelihood | -100 to +100 (50+ excellent) |
| **Response Time** | Service speed | Hours/minutes |
| **Resolution Rate** | Problem solving | % resolved |
| **Sentiment** | Overall feeling | Positive/Neutral/Negative |
| **Customer Satisfaction** | Experience quality | 1-5 or 1-10 scale |

## Reporting Framework

### Report Structure

```
1. EXECUTIVE SUMMARY
   - Key wins
   - Top metrics vs goals
   - Main insights

2. PERFORMANCE OVERVIEW
   - Reach and impressions
   - Engagement summary
   - Follower growth

3. PLATFORM BREAKDOWN
   - Facebook metrics
   - Instagram metrics
   - Other platforms

4. TOP PERFORMING CONTENT
   - Best posts
   - What worked
   - Learnings

5. UNDERPERFORMING CONTENT
   - Lowest performers
   - Why it didn't work
   - Improvements

6. AUDIENCE INSIGHTS
   - Demographics changes
   - Engagement patterns
   - Feedback themes

7. RECOMMENDATIONS
   - What to continue
   - What to change
   - Next period goals
```

### Reporting Frequency

| Report Type | Frequency | Focus |
|-------------|-----------|-------|
| **Quick check** | Daily | Engagement, issues |
| **Performance** | Weekly | Content performance, engagement |
| **Strategic** | Monthly | Goals progress, trends, recommendations |
| **Comprehensive** | Quarterly | Deep analysis, strategy review |

### Dashboard Elements

| Element | Purpose |
|---------|---------|
| **Goal tracker** | Progress toward targets |
| **Trend lines** | Performance over time |
| **Comparisons** | Period-over-period |
| **Top content** | Best performers |
| **Engagement breakdown** | By type and platform |
| **Audience growth** | Follower trends |

## Optimization Strategies

### A/B Testing Framework

| Element | What to Test |
|---------|--------------|
| **Copy** | Length, tone, CTA, emojis |
| **Visuals** | Images vs video, colors, style |
| **Timing** | Day of week, time of day |
| **Hashtags** | Number, types, specific tags |
| **Format** | Carousel vs single, Reel vs Story |

### Testing Best Practices

```
□ Test one variable at a time
□ Use sufficient sample size
□ Run tests for meaningful period
□ Document results consistently
□ Apply learnings to future content
```

### Performance Optimization Cycle

```
MEASURE
   ↓
ANALYZE
   ↓
HYPOTHESIZE
   ↓
TEST
   ↓
IMPLEMENT
   ↓
REPEAT
```

### Quick Optimization Wins

| Issue | Solution |
|-------|----------|
| Low reach | Try new content formats, Reels |
| Low engagement | Add questions, polls, CTAs |
| Low clicks | Improve copy, clearer CTAs |
| Declining followers | Increase posting frequency, value |
| Poor video performance | Improve first 3 seconds hook |

### Content Performance Checklist

```
High Performers - Analyze:
□ What format?
□ What topic/pillar?
□ What time posted?
□ What was the hook?
□ What CTA used?

Low Performers - Ask:
□ Wrong audience?
□ Wrong timing?
□ Weak hook?
□ Poor visuals?
□ Unclear message?
```
