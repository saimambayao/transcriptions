## Strategy Frameworks

### The FIERCE Framework

A comprehensive framework for strategic social media success:

| Element | Description |
|---------|-------------|
| **F - Focus** | Define social media mission distinct from company mission |
| **I - Insight** | Understand audience deeply through research |
| **E - Engage** | Build genuine connections, not just broadcasts |
| **R - Respond** | Listen and react to audience feedback |
| **C - Create** | Develop quality content aligned with goals |
| **E - Evaluate** | Measure and optimize continuously |

### Content Distribution Rules

| Rule | Breakdown | Best For |
|------|-----------|----------|
| **70/20/10** | 70% educational/entertaining, 20% curated, 10% promotional | Engagement-focused |
| **50/30/20** | 50% engagement, 30% education, 20% promotion | Balanced approach |

### Resource Allocation Rule (50/30/20)

| Allocation | Purpose |
|------------|---------|
| **50%** | Primary platform (where audience is most active) |
| **30%** | Secondary platform (growing presence) |
| **20%** | Experimental platforms (testing new channels) |

## Goal Setting

### SMART Goals Framework

| Element | Description | Example |
|---------|-------------|---------|
| **Specific** | Clear and defined | "Increase Facebook engagement" |
| **Measurable** | Quantifiable | "by 25%" |
| **Achievable** | Realistic | "with current resources" |
| **Relevant** | Aligned with objectives | "to drive store visits" |
| **Time-bound** | Has deadline | "within 3 months" |

### Common Social Media Goals

| Goal Category | Metrics to Track |
|---------------|------------------|
| **Brand Awareness** | Reach, impressions, share of voice |
| **Engagement** | Likes, comments, shares, saves |
| **Traffic** | Click-through rate, website visits |
| **Lead Generation** | Form submissions, sign-ups |
| **Sales** | Conversions, revenue from social |
| **Community** | Group members, active participants |
| **Customer Service** | Response time, resolution rate |

### Goal Alignment Hierarchy

```
ORGANIZATIONAL GOALS
        ↓
MARKETING OBJECTIVES
        ↓
SOCIAL MEDIA GOALS
        ↓
PLATFORM-SPECIFIC TARGETS
        ↓
CONTENT OBJECTIVES
```

## Platform Selection

### Platform Overview

| Platform | Primary Audience | Best Content Types | BARMM Relevance |
|----------|------------------|-------------------|-----------------|
| **Facebook** | All ages, 92% PH users | Updates, events, groups, marketplace | Dominant platform |
| **Instagram** | Millennials, Gen Z | Photos, Reels, Stories | Visual brands |
| **TikTok** | Gen Z, young millennials | Short videos, trends | Youth engagement |
| **YouTube** | All ages | Long-form video, tutorials | Educational content |
| **LinkedIn** | Professionals | Thought leadership, B2B | Partnerships |

### Platform Selection Criteria

| Factor | Questions to Ask |
|--------|------------------|
| **Audience presence** | Where does our target spend time? |
| **Content fit** | Can we create content that works here? |
| **Resources** | Do we have capacity to maintain presence? |
| **Competition** | How saturated is our space here? |
| **Features** | Does the platform offer what we need? |

## Audience Targeting

### Audience Persona Template

| Element | Description |
|---------|-------------|
| **Demographics** | Age, location, gender, income |
| **Psychographics** | Values, interests, lifestyle |
| **Behaviors** | Online habits, purchase patterns |
| **Pain Points** | Problems we can solve |
| **Goals** | What they want to achieve |
| **Preferred Platforms** | Where they spend time |
| **Content Preferences** | What they engage with |

### Audience Segmentation for BARMM

| Segment | Characteristics | Platforms | Content Approach |
|---------|----------------|-----------|------------------|
| **Young professionals** | 25-35, urban, mobile | FB, IG, TikTok | Trends, lifestyle |
| **Community members** | All ages, local focus | Facebook Groups | Updates, events |
| **Business buyers** | Decision-makers | FB, LinkedIn | Value, credibility |
| **Youth** | 18-24, students | TikTok, IG | Entertainment, education |

### Targeting Best Practices

```
□ Start with existing customer data
□ Analyze current follower demographics
□ Research competitor audiences
□ Create 2-3 distinct personas
□ Validate with actual engagement data
□ Refine based on performance
```

## Content Pillars

### What Are Content Pillars?

Content pillars are 3-5 core themes that define what your organization consistently discusses on social media.

### Content Pillar Framework

| Pillar Type | Purpose | Example Topics |
|-------------|---------|----------------|
| **Educational** | Teach and inform | How-tos, tips, explainers |
| **Inspirational** | Motivate and uplift | Success stories, quotes |
| **Entertaining** | Engage and delight | Trends, humor, behind-scenes |
| **Promotional** | Drive action | Products, offers, CTAs |
| **Community** | Build belonging | User content, events, Q&A |

### Sample Content Pillars by Sector

**Cooperative:**
1. Member success stories
2. Cooperative education (principles, benefits)
3. Product/service highlights
4. Community events
5. Financial tips and updates

**Social Enterprise:**
1. Impact stories
2. Behind-the-scenes (makers, process)
3. Product features
4. Sustainability education
5. Customer spotlights

**Government Agency:**
1. Program announcements
2. Service information
3. Success stories
4. Educational content
5. Community engagement

### Content Pillar Distribution

| Pillar | Recommended % |
|--------|---------------|
| Educational | 30-40% |
| Community/Engagement | 20-30% |
| Inspirational | 15-20% |
| Promotional | 10-15% |
| Entertainment | 10-15% |
