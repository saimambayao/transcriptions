## Naming Convention

Files follow the same `NN-descriptor.md` pattern used in the Bill Drafting Guidebook:
- `00-` prefix for front matter
- Sequential numbering `01-` through `20-` for content files
- Chapter files use `NN-chapter-XX.md` where NN is the file sequence number and XX is the chapter number
- Appendices are grouped into batches of 3 per file to keep file sizes manageable
