## File Naming Convention

Files use the `NN-descriptor.md` pattern, matching the bill-drafting-manual convention:

- `00-` prefix: front matter (TOC, about)
- `01-` prefix: introduction
- `02-` through `15-`: chapters 1-14 (offset by 1 from chapter number)
- `16-`: glossary
- `17-`: appendices
- Uppercase names: project/tracking files (PROGRESS.md)
